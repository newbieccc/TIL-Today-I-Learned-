>대문자 / 반복문 int 선언 / 인덱스 찾기 charA();

## 포인트

- str = str.toUpperCase(); / ch = Character.toUpperCase(ch);
- charAt();

```java
char ch = sc.next().charAt(0);
//Scanner를 이용하여 char형 변수를 직접 입력받을 수 없다.

//sc.next().charAt(0); -> 0번째 문자를 가져오기.
```
