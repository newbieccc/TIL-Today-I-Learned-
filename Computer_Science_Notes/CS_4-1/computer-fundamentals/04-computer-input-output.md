## 2. 컴퓨터와 데이터 입출력

### 2-1. 컴퓨터 시스템의 구성

- 컴퓨터 시스템은 기본적으로 다음 장치들로 구성됨

| 구성 요소  | 역할               |
|--------|------------------|
| 입력장치   | 데이터를 컴퓨터 내부로 입력  |
| CPU    | 명령 해석과 연산 처리     |
| 주기억장치  | 프로그램과 데이터를 임시 저장 |
| 출력장치   | 처리 결과를 사용자에게 출력  |
| 보조기억장치 | 데이터와 프로그램을 장기 저장 |

### 핵심 정리

- 입력장치를 통해 데이터가 들어오고,
  CPU와 주기억장치에서 처리된 뒤,
  출력장치나 보조기억장치로 결과가 전달됨

---

## 3. 데이터의 처리 방식과 입출력

### 3-1. 일괄처리 방식

- 데이터를 수집하였다가 일정 시점 단위로 일괄 처리하는 방식
- 예:
    - OMR 시험 답안지 채점

### 특징

- 데이터를 모아 한꺼번에 처리하므로 효율적임
- 개별 요청이 즉시 처리되지 않으므로 기다리는 시간이 필요함
- 입출력은 사람이 개입하지 않는 형식으로 이루어지는 경우가 많음
- 보조기억장치 등에 데이터를 모아서 처리한 뒤,
  보조기억장치나 프린터 등으로 처리 결과를 출력함

### 3-2. 대화식 처리 방식

- 사용자가 명령이나 데이터를 입력하면
  즉시 적절한 처리를 수행하고 사용자에게 결과를 응답하는 방식

### 예

- 웹 검색
- 문서 편집
- 게임
- 온라인 뱅킹
- 판매점 단말기(POS)

### 특징

- 사용자가 직접 데이터를 입력함
- 즉시 컴퓨터의 반응을 확인할 수 있는 출력장치가 필요함
- 사용자가 얼마나 빠르게 응답받을 수 있는지가 중요한 성능 척도임

### 핵심 정리

| 구분     | 일괄처리 방식       | 대화식 처리 방식      |
|--------|---------------|----------------|
| 처리 시점  | 일정 시점에 모아서 처리 | 입력 즉시 처리       |
| 사용자 응답 | 즉시 응답 아님      | 즉시 응답          |
| 예      | OMR 답안지 채점    | 검색, 게임, 온라인 뱅킹 |
| 중요 요소  | 처리 효율         | 응답 속도          |

---

## 4. 데이터 입력

### 4-1. 데이터 입력 방법에 따른 입력장치 분류

| 입력 방법        | 입력 매체                               |
|--------------|-------------------------------------|
| 단말기 입력       | 키보드, 마우스, 스캐너, 터치스크린, 펜 마우스, 음성입력 등 |
| 원시 데이터 직접 입력 | OMR, OCR, MICR, 바코드 등               |
| 보조기억매체       | 자기디스크, 자기테이프, 광디스크, 플래시 메모리 등       |

### 핵심 정리

- 입력장치는 데이터 입력 방식에 따라
  단말기 입력, 원시 데이터 직접 입력, 보조기억매체 입력으로 나눌 수 있음

---

## 5. 단말기 입력

### 5-1. 단말기란?

- 컴퓨터에 데이터를 입력하고,
  컴퓨터가 내보내는 데이터를 출력하는 장치
- 1950년대의 전신타자기(teletypewriter, TTY)가 초기 형태로 사용됨

### 5-2. 단순 터미널

- 극히 제한된 제어 코드 외에는 처리 능력이 없음
- 예:
    - CR
    - LF 등

### 5-3. 지능적 터미널

- 마이크로프로세서를 포함함
- 주 컴퓨터에 데이터를 보내기 전
  간단한 편집 등 일부 입력 데이터 처리를 자체적으로 수행할 수 있음

### 5-4. 현대의 단말기

- 그래픽 단말기:
    - 문자 외에도 그림을 출력할 수 있는 단말기
- 최근에는 개인용 컴퓨터가 컴퓨터 단말기로 많이 사용됨

### 5-5. 신 클라이언트와 팻 클라이언트

| 구분                   | 설명                                                             |
|----------------------|----------------------------------------------------------------|
| 신 클라이언트(thin client) | 대부분의 계산, 처리, 저장 기능을 서버가 담당하고 사용자에게 그래픽 사용자 인터페이스만 제공하는 저사양 컴퓨터 |
| 팻 클라이언트(fat client)  | 각종 응용프로그램을 내장하고 서버와 독립적인 처리 능력을 가진 컴퓨터                         |

### 핵심 정리

- 단말기는 입력과 출력을 모두 담당할 수 있는 장치임
- 신 클라이언트는 서버 의존도가 높고,
  팻 클라이언트는 자체 처리 능력이 큼

---

## 6. 주요 단말기 입력장치

### 6-1. 키보드

- 문자나 기호를 입력하기 위한 기본적인 입력장치
- 타자기의 배열을 차용한 키 배열을 사용함

### 영문 키보드

- 쿼티(QWERTY)
- 드보락(Dvorak)

### 한글 키보드

- 두벌식
- 세벌식
- 1982년에 KS X 5002 표준 두벌식 키보드를 PC 표준 키보드로 확정함

### 핵심 정리

- 키보드는 문자와 기호를 입력하는 대표적인 장치임
- 대부분의 컴퓨터에서는 쿼티 키보드가 사용됨

---

### 6-2. 마우스

- 평면상에서의 상대적 움직임을 감지하여
  좌표와 같은 공간 정보를 입력할 수 있는 포인팅 장치

### 기계식 마우스

- 마우스 안의 작은 공이 구르는 방향을
  축과 축 둘레의 센서를 통해 읽어 내는 방식

### 광학식 마우스

- LED, 레이저 등의 조명을 바닥면에 비추고
  연속적인 이미지를 센서로 읽어 이동을 계산하는 방식

### 마우스 조작

- 클릭(click)
    - 한 번 누르기
- 더블 클릭(double click)
    - 두 번 누르기
- 끌기(drag)
- 끌어서 놓기(drag-and-drop)

### 핵심 정리

- 마우스는 화면의 좌표와 위치를 선택하는 대표적인 포인팅 장치임
- 마우스를 통해 선택, 실행, 이동, 복사 등의 동작을 지시할 수 있음

---

### 6-3. 터치스크린

- 컴퓨터 화면의 접촉을 인지하여 좌표를 입력하는 장치
- 손가락이나 스타일러스와 같은 도구로 화면을 접촉함

### 입력 방식

- 정전식
- 감압식
- 광학식 등

### 용도

- 모바일 단말기
- 대화형 키오스크(kiosk) 등

### 핵심 정리

- 터치스크린은 화면 자체가 입력장치 역할을 함
- 모바일 환경에서 가장 많이 사용되는 입력 방식 중 하나임

---

### 6-4. 그래픽 태블릿

- 종이에 그리듯 좌표나 펜의 궤적을 입력할 수 있는 장치
- 스타일러스를 이용하여 절대 위치를 입력할 수 있음
- 스타일러스의 압력을 이용해 획의 굵기 표현 등도 가능함

### 용도

- 컴퓨터 그래픽스
- CAD
- 필기 입력
- 마우스 대용 등

### 핵심 정리

- 그래픽 태블릿은 정밀한 좌표 입력과 그림 입력에 적합함

---

## 7. 원시 데이터의 직접 입력

### 7-1. 기계가 읽을 수 있는 원시문서

- 원시 데이터를 사람이 다시 입력하지 않고
  기계가 직접 읽어들이는 방식

### 대표 방식

| 방식   | 의미               |
|------|------------------|
| OMR  | 광학마크 인식          |
| OCR  | 광학문자 인식          |
| MICR | 자기잉크문자 인식        |
| 바코드  | 흑백 막대로 문자나 숫자 표현 |

---

### 7-2. 광학마크 인식(OMR)

- OMR: Optical Mark Recognition
- 정해진 위치에 표시한 마크를 인식함
- 예:
    - 시험 답안지
    - 설문지

### 핵심 정리

- OMR은 정해진 위치에 표시된 마크를 읽는 방식임

---

### 7-3. 광학문자 인식(OCR)

- OCR: Optical Character Recognition
- 인쇄되거나 손으로 쓴 글자를 인식하여 입력함
- 스캐너나 카메라를 통해 입력된 이미지에서 문자를 판독함

### 핵심 정리

- OCR은 이미지 속 문자를 인식하는 방식임
- 인쇄 문자나 손글씨를 컴퓨터 데이터로 바꿀 때 사용됨

---

### 7-4. 자기잉크문자 인식(MICR)

- MICR: Magnetic Ink Character Recognition
- 자기 잉크로 인쇄된 글자를 인식함
- 글자 위에 필기나 낙인 등이 있더라도 인식이 가능함
- 대표 글꼴:
    - E-13B 서체 집합

### 핵심 정리

- MICR은 자기 잉크로 인쇄된 문자를 읽는 방식임
- 금융권 수표 처리 등에서 활용됨

---

### 7-5. 바코드

- 흑과 백의 막대로 문자나 숫자를 표현함
- 상품, 도서 등 다양한 지질에 인쇄하여 사용 가능함
- 예:
    - 세계상품코드(UPC)
    - UPC-A 바코드

### 핵심 정리

- 바코드는 상품 정보나 식별 정보를 빠르게 입력하기 위한 대표적인 자동 입력 기술임

---

## 8. 기타 컴퓨터 입력

### 8-1. 판매점 단말기(POS)

- POS: Point of Sale
- 고객과 거래가 일어나는 시간과 장소에서
  즉시 컴퓨터 시스템에 데이터를 입력함

### 역할

- 판매기록 유지
- 재고량 확인
- 판매에 따른 세금 계산
- 수표 조회
- 기타 사업 운영에 필요한 작업 자동 수행

### 핵심 정리

- POS는 판매 현장에서 거래 데이터를 즉시 입력하고 처리하는 단말기임

---

### 8-2. 이미지 스캐너와 디지털 카메라

- 영상 데이터를 입력하는 장치
- 종이 문서나 물체, 화면 등을 이미지 데이터로 변환함

### 핵심 정리

- 이미지 스캐너와 디지털 카메라는 시각 정보를 컴퓨터로 입력하는 장치임

---

### 8-3. 매트릭스 코드

- 2차원 바코드 또는 2차원 코드라고도 함
- 2차원 영역에 패턴을 나열하여 정보를 표현함

### QR 코드

- 1994년 일본의 덴소 웨이브에서 개발
- 버전 40-L의 경우 최대 2,953바이트 저장 가능
- URL, 주소, 문자 정보 등을 담을 수 있음

### 핵심 정리

- QR 코드는 대표적인 2차원 코드임
- 일반 바코드보다 더 많은 정보를 저장할 수 있음

---

### 8-4. RFID

- RFID: Radio-Frequency Identification
- IC칩과 전자기장을 이용한 무선 데이터 전송 기술을 사용함
- 안테나와 집적회로로 구성된 RFID 태그를 사용함

### 특징

- 태그 안에 정보를 저장하거나 수정할 수 있음
- 여러 개의 태그를 동시에 구분할 수 있음

### 핵심 정리

- RFID는 무선으로 태그 정보를 인식하는 기술임
- 물류, 출입관리, 재고관리 등에서 활용 가능함

---

### 8-5. 보조기억장치를 통한 입력

- 데이터를 자기디스크, 광디스크, 플래시메모리 등의 보조기억장치에 저장한 뒤
  이 데이터를 입력으로 사용함

### 핵심 정리

- 저장매체에 기록된 데이터를 다시 읽어 컴퓨터 입력으로 사용할 수 있음

---

## 9. 입력 데이터 편집

### 9-1. 입력 데이터 편집의 필요성

- GIGO:
    - Garbage in, garbage out
- 입력된 데이터가 정확하지 않으면 처리 결과도 쓸모가 없음
- 따라서 정확한 데이터가 입력되도록 검증하는 과정이 필요함

### 핵심 정리

- 잘못된 데이터가 입력되면 올바른 결과를 얻을 수 없음
- 입력 단계에서 데이터 검증이 중요함

---

## 10. 입력 데이터의 정확성 확인을 위한 검사

### 10-1. 문자 종류 확인

- 입력된 자료의 유형이 올바른지 검사함
- 예:
    - 우편번호 입력 시 숫자만 입력되었는지 확인

### 10-2. 값의 범위 확인

- 입력된 값이 허용된 범위 안에 있는지 확인함
- 예:
    - 나이, 점수, 날짜 등

### 10-3. 값의 타당성 확인

- 입력값이 실제 의미상 가능한 값인지 확인함
- 예:
    - 존재하지 않는 날짜
    - 잘못된 코드 값 등

### 10-4. 일관성 확인

- 여러 입력값 사이에 모순이 없는지 확인함
- 예:
    - 생년월일과 나이의 불일치
    - 우편번호와 주소의 불일치 등

### 10-5. 체크디지트(check digit)

- 데이터 입력 과정에서 발생하는 오류를 검사하기 위해 추가된 숫자
- 오류의 예:
    - 숫자가 다른 값으로 바뀌는 경우
    - 숫자의 순서를 뒤바꾸어 입력한 경우 등
- 데이터에 미리 정해 놓은 연산을 통해 얻은 체크디지트를
  실제 데이터와 함께 입력하거나 전송함

### 예시

- 학번 저장 예시:
    - 원래 학번:
        - `202534-198766`
    - 체크디지트를 계산하여 뒤에 `1`을 붙임
    - 저장 학번:
        - `202534-1987661`

### 핵심 정리

- 입력 데이터 편집은 오류를 미리 발견하기 위한 과정임
- 체크디지트는 입력 오류 검출을 쉽게 하기 위한 추가 숫자임

## 11. 정보의 출력

## 12. 디스플레이 모니터

### 12-1. 특징

- 가장 일반적으로 사용되는 컴퓨터 출력장치
- 컴퓨터와 대화하듯 사용자의 입력에 반응하여
  출력되는 정보를 동적으로 제공할 수 있음
- 초기에는 CRT 모니터가 사용되었고,
  이후 LCD, OLED 등의 평판형 모니터가 보급됨

### 12-2. 출력 방식

- 화면은 가로와 세로의 격자 형태로 촘촘히 나열된 점으로 구성됨
- 각 점은 픽셀 또는 화소라고 부름
- 여러 가지 색과 밝기로 정보를 출력함
- 이러한 방식은 래스터 방식임

### 12-3. 화면비율

- 4:3
- 16:9

### 12-4. 해상도

| 구분      | 해상도       |
|---------|-----------|
| HD      | 1280×720  |
| Full HD | 1920×1080 |
| QHD     | 2560×1440 |
| 4K UHD  | 3840×2160 |
| 8K UHD  | 7680×4320 |

### 12-5. 플렉시블 디스플레이

- 유연하게 휘거나 접을 수 있는 디스플레이
- 전자종이나 OLED를 활용함

### 핵심 정리

- 모니터는 대표적인 영상 출력장치임
- 해상도는 화면을 구성하는 픽셀 수를 의미함
- 플렉시블 디스플레이는 휘거나 접을 수 있는 차세대 디스플레이 기술임

## 13. 프린터

### 13-1. 출력 방식

#### 충격인쇄

- 종이, 리본, 글자를 함께 부딪치게 하여
  리본의 잉크를 종이에 찍어 내는 방식

### 예

- 데이지 휠 프린터
- 도트 매트릭스 프린터
- 라인 프린터 등

#### 비충격인쇄

- 충격 없이 인쇄하는 방식

### 예

- 레이저 프린터
- 잉크젯 프린터
- 감열식 프린터 등

### 핵심 정리

- 프린터는 인쇄 방식에 따라 충격인쇄와 비충격인쇄로 나눌 수 있음
- 현대에는 레이저, 잉크젯 등 비충격인쇄 방식이 많이 사용됨

## 14. 기타 컴퓨터 출력장치

### 14-1. 플로터

- 선 그리기 형태의 그림을 출력하기 위한 장치
- 펜 플로터는 다양한 색의 펜과 원통 또는 평면 종이판을 사용함
- 현재는 고품질 출력을 위해
  잉크젯 방식의 광폭 프린터를 이용해 대형 출력물을 만드는 경우가 많음

### 14-2. 비디오 프로젝터

- 스크린에 슬라이드나 동영상 같은 영상 출력을 투사하는 장치
- 큰 화면을 출력할 수 있어 다음 용도로 많이 사용됨
    - 회의실
    - 강의용
    - 홈시어터

### 14-3. 헤드 마운티드 디스플레이

- 작은 광학 디스플레이 장치를 통해 영상을 두 눈에 전달할 수 있도록
  머리에 쓰는 디스플레이 장치
- 3차원 입체 장면으로 가상현실 환경을 보여 줌
- 실제계와 사이에 컴퓨터가 생성한 정보를 겹쳐서 보여 줄 수 있음

### 14-4. 3차원 프린터

- 컴퓨터를 이용하여 설계한 3차원 입체 모델을
  실제 물체로 제작하는 장치

### 제작 방식

| 방식  | 설명                                                             |
|-----|----------------------------------------------------------------|
| 절삭형 | 컴퓨터 수치제어 공작기계와 같은 도구를 이용하여 재료를 깎아 원하는 물체를 만드는 방식               |
| 적층형 | 석고나 나일론 가루, 플라스틱 액체나 필라멘트 같은 재료를 미세한 단위로 한 층씩 쌓아 올려 물체를 만드는 방식 |

### 3D 프린터 제작 과정

1. 모델링
2. 프린팅
3. 마무리

### 관련 도구

- CAD
- 3차원 스캐너
- 연마, 도색, 조립 등

### 핵심 정리

- 플로터는 선 그림 출력에 적합함
- 비디오 프로젝터는 큰 화면 출력에 적합함
- 헤드 마운티드 디스플레이는 VR·AR 환경에 활용됨
- 3D 프린터는 디지털 모델을 실제 물체로 만드는 장치임

## 17. 정리하기

1. 대화식 처리에서는 주로 키보드, 마우스 등의 입력장치와
   디스플레이 모니터가 포함된 형태의 컴퓨터 단말기를 사용하여
   온라인 상태에서 사용자가 직접 입력함

2. 좌표와 같은 공간정보를 입력할 수 있는 장치에는
   마우스, 터치스크린, 그래픽 태블릿 등이 있음

3. 기계가 읽을 수 있는 원시문서에는
   OMR, OCR, MICR, 바코드 등이 있음

4. 입력 데이터는 데이터의 자료형, 값의 타당성, 값의 범위 적정성,
   일관성 편집 과정 등을 통해 올바른 데이터가 입력되도록 함

5. 디스플레이 모니터는 대화식 처리에서 가장 많이 사용되는 대표적 출력장치임
    - LCD, PDP, OLED 등의 평판형 모니터가 주로 사용됨

6. 헤드 마운티드 디스플레이는 3차원 입체 장면으로
   가상현실 환경을 보여 주거나 실제계에 컴퓨터가 생성한 정보를 겹쳐 보여 주기 위해 사용됨

7. 미래에는 음성, 필기, 동작 등을 사용하여 편하게 입력하고,
   사용자가 정보를 쉽고 직관적으로 이해할 수 있는 출력장치가 실용적으로 사용될 수 있음

8. 3차원 프린터와 같은 출력 기술은 제조업에 영향을 미칠 수 있는 중요한 기술로 인식되고 있음

## 19. 연습문제 정리

### Q1. 대화식처리 시스템에 대한 올바른 설명은?

1. 사용자의 개별 요청은 즉시 처리되지 않는다.
2. 데이터를 수집하여 한꺼번에 처리한다.
3. 사용자에게 얼마나 빠르게 응답할 수 있는가가 중요한 성능 척도이다.
4. 오프라인 형식으로 입력 및 출력을 한다.

- 정답: **3**

### 해설

- 대화식처리는 사용자가 명령이나 데이터를 입력하면
  즉시 적절한 처리를 수행하고 결과를 응답하는 방식임
- 따라서 사용자에게 얼마나 빠르게 응답할 수 있는지가 중요한 성능 척도임

---

### Q2. 대부분의 계산 및 처리, 저장 기능을 서버가 담당하고,

사용자에게 그래픽 사용자 인터페이스만 제공하는 필수 요소만으로 구성된 저사양 컴퓨터 형태의 단말기는?

1. 신 클라이언트
2. 단순 터미널
3. 지능적 터미널
4. 전신 타자기

- 정답: **1**

### 해설

- 신 클라이언트(thin client)는 대부분의 계산과 저장을 서버가 담당함
- 사용자에게는 그래픽 사용자 인터페이스만 제공하는 저사양 단말기임

---

### Q3. 인쇄되거나 손으로 쓴 글자를 이미지 스캐너로 입력하여 인식하는 문자 입력 방식은?

1. OMR
2. MICR
3. OCR
4. 바코드

- 정답: **3**

### 해설

- OCR은 Optical Character Recognition의 약자임
- 인쇄 문자나 손글씨를 인식하여 컴퓨터에 입력하는 방식임

### Q4. 데이터 입력 시 체크디지트의 용도는 무엇인가?

1. 데이터 입력 과정에서 오류가 발생하였는지를 검사하기 위한 숫자이다.
2. 입력할 값이 준비되었는지 검사하기 위한 것이다.
3. 입력된 여러 값 사이의 모순이 존재하는지 검사하기 위한 것이다.
4. 데이터가 입력되었음을 알리기 위한 값이다.

- 정답: **1**

### 해설

- 체크디지트는 입력 과정에서 숫자가 잘못 입력되었는지,
  순서가 바뀌었는지 등을 검사하기 위해 추가하는 숫자임

## 20. 시험 직전 체크포인트

- 컴퓨터 시스템은 입력장치, CPU, 주기억장치, 출력장치, 보조기억장치로 구성된다.
- 일괄처리는 데이터를 모아 일정 시점에 한꺼번에 처리하는 방식이다.
- 대화식 처리는 사용자 입력에 즉시 응답하는 처리 방식이다.
- 대화식 처리에서는 응답 속도가 중요하다.
- 단말기는 입력과 출력 기능을 함께 수행할 수 있다.
- 신 클라이언트는 서버가 대부분의 처리와 저장을 담당한다.
- 팻 클라이언트는 자체 처리 능력을 가진다.
- 키보드는 문자와 기호를 입력하는 기본 장치이다.
- 마우스는 좌표 정보를 입력하는 포인팅 장치이다.
- 터치스크린은 화면 접촉을 통해 좌표를 입력한다.
- 그래픽 태블릿은 펜의 위치와 궤적을 입력한다.
- OMR은 정해진 위치의 마크를 인식한다.
- OCR은 인쇄 문자나 손글씨를 인식한다.
- MICR은 자기 잉크로 인쇄된 문자를 인식한다.
- 바코드는 흑백 막대로 문자나 숫자를 표현한다.
- QR 코드는 2차원 코드이며 바코드보다 많은 정보를 저장할 수 있다.
- RFID는 무선으로 태그 정보를 인식한다.
- GIGO는 잘못된 입력이 잘못된 출력을 만든다는 의미이다.
- 체크디지트는 데이터 입력 오류를 검사하기 위한 추가 숫자이다.
- 디스플레이 모니터는 대표적인 영상 출력장치이다.
- 해상도는 화면을 구성하는 픽셀 수를 의미한다.
- OLED는 플렉시블 디스플레이에 활용될 수 있다.
- 프린터는 충격인쇄와 비충격인쇄 방식으로 나눌 수 있다.
- 3D 프린터는 3차원 모델을 실제 물체로 제작하는 장치이다.
- 미래 입출력은 음성, 필기, 동작, 센서, 컴퓨터비전 중심으로 발전한다.
