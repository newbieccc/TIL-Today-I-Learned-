## 1. 모바일 서비스 개요

### 1.1 모바일의 의미

모바일은 정보통신에서 이동성을 가진 것들을 총칭하는 말이다.

즉, “움직일 수 있는” 정보통신 환경과 기기를 의미한다.

| 구분     | 설명                     |
|--------|------------------------|
| 모바일    | 정보통신에서 이동성을 가진 것       |
| 모바일 장치 | 이동하면서 사용할 수 있는 장치      |
| 예      | 휴대폰, PDA, 스마트폰, 태블릿 PC |

---

### 1.2 모바일 장치

모바일 장치는 휴대폰이나 휴대용 개인정보 단말기처럼 이동하면서 사용할 수 있는 장치이다.

대표적인 모바일 장치는 다음과 같다.

- 휴대폰
- PDA
- 스마트폰
- 태블릿 PC
- 아이팟 터치
- 아이패드

모바일 장치는 가볍고 작아 이동성이 좋지만, 디스플레이가 작고 전력 공급이 제한적이라는 특징이 있다.

---

### 1.3 PDA

PDA는 Personal Digital Assistants의 약자이다.

작은 데스크톱, 휴대용 컴퓨터의 성격을 가지며 다음과 같은 기능을 제공하였다.

| 기능       | 설명             |
|----------|----------------|
| 무선 인터넷   | 이동 중 인터넷 접속    |
| 문서 작업    | 간단한 문서 작성 및 관리 |
| 전자상거래    | 온라인 거래 가능      |
| 온라인 주식거래 | 모바일 금융·투자 기능   |
| 개인정보 관리  | 일정, 주소록 등 관리   |

PDA는 이후 스마트폰 발전의 기반이 되었다.

---

## 2. 모바일 시장의 패러다임 변화

### 2.1 음성 중심에서 모바일 서비스 중심으로

초기의 모바일 시장은 음성통화 중심이었다.

하지만 스마트폰이 등장하면서 데이터, 애플리케이션, 운영체제, 소프트웨어 업체가 중요해지는 모바일 서비스 중심 시장으로 변화하였다.

| 구분      | 음성 중심              | 모바일 서비스 중심                   |
|---------|--------------------|------------------------------|
| 서비스     | 음성통화, 데이터          | 애플리케이션, 데이터, 음성통화            |
| 휴대폰 경쟁력 | 하드웨어 기능            | 애플리케이션, 운영체제, 하드웨어 기능        |
| 산업 구조   | 단말 제조사, 이동통신사, 소비자 | 단말 제조사, 이동통신사, 소비자, 소프트웨어 업체 |
| 핵심 가치   | 통화 품질과 단말 기능       | 앱 생태계와 서비스 경험                |

---

### 2.2 패러다임 변화의 의미

스마트폰 시대에는 단말기 자체의 성능뿐 아니라 운영체제와 애플리케이션 생태계가 중요해졌다.

즉, 모바일 산업의 중심이 다음과 같이 이동하였다.

    하드웨어 중심 → 운영체제 중심 → 애플리케이션·서비스 중심

## 3. 스마트폰

### 3.1 스마트폰의 의미

스마트폰은 휴대폰에 PDA 기능이 추가된 지능형 단말기이다.

통화와 문자뿐 아니라 인터넷 통신, 정보 검색, 문서 서비스 등 컴퓨터에서 할 수 있는 대부분의 기능을 수행할 수 있다.

| 특징      | 설명                 |
|---------|--------------------|
| 지능형 단말기 | 휴대폰 + PDA + 컴퓨터 기능 |
| 인터넷 통신  | 웹 검색, 메일, 메신저 사용   |
| 정보 검색   | 위치, 뉴스, 자료 검색      |
| 문서 서비스  | 문서 작성과 확인          |
| 앱 설치    | 사용자가 앱을 설치·삭제 가능   |

---

### 3.2 스마트폰의 등장

2007년 애플의 아이폰에서 ‘스마트’라는 용어가 본격적으로 사용되었다.

스마트폰은 수백 종의 애플리케이션을 사용자가 직접 설치하고 삭제할 수 있으며, 수백만 개의 애플리케이션이 앱스토어를 통해 유통된다.

---

### 3.3 세계 스마트폰 시장 규모

세계 스마트폰 시장은 스마트폰 등장 이후 급격히 성장하였다.

| 시기       |         출하량 |
|----------|------------:|
| 2009년 초기 |  약 1억 8천만 대 |
| 2021년    | 약 13억 8천만 대 |
| 2023년    | 약 11억 7천만 대 |

## 4. 모바일 운영체제

### 4.1 운영체제의 의미

운영체제(OS, Operating System)는 시스템 소프트웨어이다.

운영체제는 하드웨어와 응용프로그램 사이에서 중재 역할을 하며, 컴퓨터나 모바일 장치를 사용할 수 있게 해 준다.

| 기능           | 설명                    |
|--------------|-----------------------|
| 하드웨어 관리      | CPU, 메모리, 저장장치 등 관리   |
| 프로그램 실행 관리   | 응용프로그램 실행 지원          |
| 사용자 인터페이스 제공 | 사용자가 장치를 쉽게 조작하도록 지원  |
| 파일 시스템 관리    | 파일 저장, 삭제, 접근 관리      |
| 입출력 관리       | 키보드, 터치, 화면, 스피커 등 제어 |
| 메모리 할당       | 프로그램 실행에 필요한 메모리 관리   |

---

### 4.2 모바일 운영체제의 종류

모바일 소프트웨어 플랫폼에는 다음과 같은 종류가 있다.

| 운영체제          | 개발사 또는 관련 기관       |
|---------------|--------------------|
| iOS           | 애플                 |
| Android       | 구글                 |
| KaiOS         | KaiOS Technologies |
| Windows Phone | 마이크로소프트            |
| Symbian       | 노키아                |
| BlackBerry    | RIM                |
| LiMo          | LiMo Foundation    |

---

### 4.3 모바일 운영체제의 특징

모바일 운영체제는 모바일 장치의 특성에 맞게 설계된다.

| 특징        | 설명                         |
|-----------|----------------------------|
| 단순성       | 사용자가 쉽게 사용할 수 있어야 함        |
| 무선통신 지원   | 광대역 및 지역 무선통신 지원           |
| 멀티미디어 지원  | 모바일 멀티미디어 포맷 처리            |
| 다양한 입력 방식 | 터치, 음성, 센서 등               |
| 장치 제어     | 스마트폰, PDA, 태블릿 PC, 정보기기 제어 |
| 앱 관리      | 프로그램 내려받기, 업데이트, 저장, 삭제 가능 |

---

### 4.4 BREW와 WIPI

| 구분   | 설명                                              |
|------|-------------------------------------------------|
| BREW | Binary Runtime Environment for Wireless         |
| 개발사  | 퀄컴                                              |
| 특징   | 휴대폰용 폐쇄형 플랫폼                                    |
| WIPI | Wireless Internet Platform for Interoperability |
| 특징   | 2005년~2009년 모든 국내 휴대폰에 의무 탑재                    |

WIPI는 공통된 인터페이스를 통해 휴대폰 애플리케이션 제작을 쉽게 하기 위한 플랫폼이었다.

---

### 4.5 모바일 운영체제 분류

모바일 운영체제는 공개형과 공개소스형으로 구분할 수 있다.

| 구분    | 설명            | 예                      |
|-------|---------------|------------------------|
| 공개형   | API를 개발자에게 제공 | 심비안, 윈도 모바일, iOS, 블랙베리 |
| 공개소스형 | 소스코드를 공개      | 리눅스 기반 안드로이드           |

공개소스형은 애플리케이션 개발 업체와 단말기 제조업체가 개발 비용을 줄이고 다양한 콘텐츠와 애플리케이션을 확보하기 쉽게 해 준다.

---

### 4.6 모바일 운영체제의 영향

모바일 운영체제는 모바일 생태계의 핵심이다.

| 영향           | 설명                               |
|--------------|----------------------------------|
| 수익 구조 변화     | 기존 모바일 소프트웨어 개발업체의 수익 구조 변화      |
| 앱·콘텐츠 의존도 증가 | 모바일 장치에서 애플리케이션과 콘텐츠의 중요성 증가     |
| 이동통신 사업 변화   | 이동통신 사업의 구심점 역할                  |
| 모바일 생태계 형성   | 앱 개발자, 플랫폼, 제조사, 사용자가 연결된 생태계 형성 |

## 5. 모바일 운영체제의 종류와 특징

## 5.1 안드로이드

### 5.1.1 안드로이드의 개요

안드로이드는 구글이 모바일 관련 업체들과 OHA를 결성한 뒤 2007년에 발표한 모바일 운영체제이다.

운영체제, 미들웨어, 자바 기반 애플리케이션을 포함하며, 소스의 대부분을 개방하였다.

| 구분    | 내용                         |
|-------|----------------------------|
| 개발 주체 | 구글                         |
| 발표    | 2007년                      |
| 협력체   | OHA, Open Handset Alliance |
| 기반    | 리눅스 커널                     |
| 특징    | 오픈소스 지향                    |

---

### 5.1.2 안드로이드의 특징

| 특징     | 설명                     |
|--------|------------------------|
| 리눅스 기반 | 운영체제 커널이 리눅스로 되어 있음    |
| 자바 개발  | 자바로 응용프로그램 개발 가능       |
| 오픈소스   | 많은 정보가 공개되어 적용이 쉬움     |
| 개발도구   | 안드로이드 스튜디오 무료 제공       |
| 구글 연동  | 구글 클라우드 서비스와 손쉽게 연동    |
| 앱 유통   | 플레이스토어를 통해 앱 공개와 판매 가능 |

---

### 5.1.3 안드로이드 스마트폰

안드로이드를 탑재한 스마트폰은 2009년부터 출시되었다.

대표적인 안드로이드 스마트폰은 다음과 같다.

- 삼성 갤럭시
- 샤오미
- 화웨이

2022년 기준 안드로이드 점유율은 약 71%로 소개된다.

---

## 5.2 iOS

### 5.2.1 iOS의 개요

iOS는 애플 제품 중 주로 모바일 기기에 탑재되는 운영체제이다.

초기에는 아이폰, 아이패드, 아이팟 터치, Apple TV에 사용되었다.

| 구분        | 내용                         |
|-----------|----------------------------|
| 개발사       | 애플                         |
| 초기 명칭     | iPhone OS                  |
| 명칭 변경     | 2010년 iOS 4 발표와 함께 iOS로 명명 |
| 2024년 버전  | iOS 18.0                   |
| iPad 운영체제 | 2019년부터 iPadOS             |

---

### 5.2.2 iOS의 특징

| 특징     | 설명                                    |
|--------|---------------------------------------|
| 직관적 UI | 학습 과정 없이 쉽게 사용할 수 있음                  |
| 앱스토어   | 다양한 앱과 콘텐츠 보유                         |
| 보안성    | 애플이 하드웨어와 소프트웨어를 통합 관리하여 상대적으로 보안에 강함 |
| 동일 SDK | 아이폰, 아이패드 등 여러 기기용 앱 개발 가능            |

2022년 기준 iOS 점유율은 약 28%로 소개된다.

---

## 5.3 KaiOS

KaiOS는 KaiOS Technologies사가 2017년 3월 Firefox OS를 기반으로 개발한 운영체제이다.

저사양, 저전력 스마트 피처폰에 맞게 경량화되었다.

| 특징    | 설명                        |
|-------|---------------------------|
| 기반    | Firefox OS                |
| 대상    | 저사양, 저전력 스마트 피처폰          |
| 지원 기능 | 4G, 와이파이, 블루투스, 음성인식, GPS |
| 설치 용량 | 약 200MB                   |
| 확산 지역 | 인도 등 개발도상국 중심             |

---

| 구분    | 내용                                 |
|-------|------------------------------------|

| 공개    | 2010년 2월 MWC10                     |
| 주요 버전 | Windows Phone 8.1                  |
| 통합 시도 | 2015년 PC 및 모바일에서 가능한 Windows 10 출시 |

## 5.5 BlackBerry

BlackBerry는 2002년 RIM이 자체 개발한 블랙베리 OS를 기반으로 한다.

초기 미국에서는 이메일 송수신이 편리하여 업무용으로 인기가 있었다.

| 특징  | 설명                                                        |
|-----|-----------------------------------------------------------|
| 개발사 | RIM                                                       |
| 출시  | 2002년                                                     |
| 장점  | 이메일 송수신 편리                                                |
| 용도  | 업무용 스마트폰                                                  |
| 지원  | 멀티태스킹, MIDP, WAP, Exchange Server, 이메일 시스템, GroupWise 동기화 |

- 모바일 영화
- 음성통화
- 문자 메시지

---

### 6.2 스마트폰 서비스

스마트폰은 모바일 서비스를 크게 확장하였다.

| 증강현실      | 카메라와 센서를 활용한 AR 서비스      |

### 6.3 스마트폰에서 요구되는 기능

스마트폰에는 여러 기능이 통합된다.

| 기능               | 설명           |
|------------------|--------------|

## 9. 웨어러블 기기

### 9.1 웨어러블 기기의 의미

웨어러블 기기(Wearable Device)는 웨어러블 컴퓨터라고도 부른다.

안경, 시계, 의복 등과 같이 착용할 수 있는 형태의 컴퓨터이다.

| 구분    | 내용                    |
|-------|-----------------------|
| 원어    | Wearable Device       |
| 다른 표현 | Wearable Computer     |
| 형태    | 안경, 시계, 의복, 신발 등      |
| 연결    | 스마트폰이나 태블릿과 무선 연동     |
| 위치    | 사용자 신체의 가장 가까운 곳에서 사용 |

---

### 9.2 웨어러블 기기의 특징

웨어러블 기기는 사용자의 신체 가까이에서 정보를 수집하고, 스마트폰 또는 태블릿과 연동하여 소통할 수 있다.

| 특징    | 설명                        |
|-------|---------------------------|
| 착용성   | 몸에 착용하는 형태                |
| 무선 연동 | 스마트폰, 태블릿과 연결             |
| 센서 활용 | 건강, 운동, 위치 등 정보 수집        |
| 실시간성  | 사용자 상태를 실시간으로 확인          |
| 개인화   | 개인의 신체·활동 정보를 기반으로 서비스 제공 |

---

### 9.3 웨어러블 기기 예

대표적인 웨어러블 기기는 다음과 같다.

- 스마트워치
- 스마트 안경
- 스마트 의복
- 스마트 신발
- 피트니스 밴드
- 헬스케어 기기

강의에서는 갤럭시 Watch와 글로벌 웨어러블 기술 시장 규모가 소개된다.

## 정리하기

1. 모바일은 정보통신에서 이동성을 가진 것들을 총칭하는 말이다.

2. 모바일 장치는 휴대폰, PDA, 스마트폰, 태블릿 PC처럼 이동하면서 사용할 수 있는 장치이다.

3. PDA는 Personal Digital Assistants의 약자로, 작은 데스크톱 또는 휴대용 컴퓨터 성격을 가지며 무선 인터넷, 문서 작업, 전자상거래 등의 기능을 제공하였다.

4. 모바일 시장은 음성 중심에서 모바일 서비스 중심으로 변화하였다.

5. 스마트폰은 휴대폰에 PDA 기능이 추가된 지능형 단말기로, 인터넷 통신, 정보 검색, 문서 서비스 등 컴퓨터에서 할 수 있는 대부분의 기능을 제공한다.

6. 스마트폰에서는 사용자가 수많은 애플리케이션을 직접 설치하고 삭제할 수 있다.

7. 운영체제는 하드웨어 관리, 프로그램 실행 관리, 사용자 인터페이스, 파일 시스템 관리 등을 담당하는 시스템 소프트웨어이다.

8. 모바일 운영체제에는 iOS, Android, KaiOS, Windows Phone, Symbian, BlackBerry, LiMo 등이 있다.

9. 모바일 운영체제는 프로그램 내려받기, 업데이트, 저장, 삭제, 이메일, 메시지 등의 기능을 지원한다.

10. 모바일 운영체제는 공개형과 공개소스형으로 나눌 수 있다.

11. 안드로이드는 리눅스 기반의 오픈소스 지향 모바일 운영체제이며, 자바로 응용프로그램 개발이 가능하다.

12. iOS는 애플의 모바일 운영체제로, 직관적인 UI와 앱스토어, 상대적으로 뛰어난 보안성을 특징으로 한다.

13. KaiOS는 저사양·저전력 스마트 피처폰에 맞게 경량화된 운영체제이다.

14. Windows Phone은 마이크로소프트가 개발한 모바일 기기용 운영체제이다.

15. BlackBerry는 이메일 송수신이 편리하여 초기 미국에서 업무용으로 인기를 얻었다.

16. 스마트폰은 모바일 비즈니스, 모바일 마케팅, 멀티미디어 서비스, 오피스 문서 처리, 일정관리, 증강현실 등 다양한 서비스를 제공한다.

17. 앱스토어는 모바일 애플리케이션을 자유롭게 사고팔 수 있는 온라인 장터이다.

18. 카카오톡은 스마트폰에서 무선이동통신을 이용하여 무료 채팅을 가능하게 하는 애플리케이션이다.

19. 웨어러블 기기는 안경, 시계, 의복 등 착용할 수 있는 형태의 컴퓨터이다.

20. 웨어러블 기기는 사용자 신체 가까이에서 스마트폰이나 태블릿과 무선으로 연동하여 소통할 수 있다.

## 연습문제 정리

### Q1. 휴대폰에 PDA 기능이 추가되어 통화, 문서 서비스와 컴퓨터에서 할 수 있는 인터넷 통신, 정보검색 등의 대부분 기능을 추가한 지능형 단말기를 무엇이라고 하는가?

1. 인터넷 전화
2. 스마트폰
3. 3G통신
4. 노트북

정답: 2

해설:
스마트폰은 휴대폰에 PDA 기능이 추가된 지능형 단말기이다. 통화뿐 아니라 인터넷 통신, 정보 검색, 문서 서비스 등 컴퓨터에서 할 수 있는 대부분의 기능을 제공한다.

---

### Q2. 리눅스 기반의 오픈소스 방식 모바일 운영체제로, 자바 언어로 개발되는 애플리케이션들을 포함하고 있는 것은?

1. 윈도폰
2. iOS
3. 블랙베리
4. 안드로이드

정답: 4

해설:
안드로이드는 리눅스 기반의 오픈소스 지향 모바일 운영체제이며, 자바로 응용프로그램 개발이 가능하다.

---

### Q3. 다음 중 모바일 운영체제의 특징이 아닌 것은?

1. 모바일 운영체제는 단순하고, 유선 연결을 통해서만 인터넷이 가능하다.
2. 모바일 운영체제는 모바일 장치나 정보 기기를 제어하기 위한 것이다.
3. 임베디드 시스템 및 무선 장치를 모바일 운영체제로 구동 가능하다.
4. BREW는 폐쇄형 플랫폼으로 휴대폰에서 사용하였다.

정답: 1

해설:

### Q4. 다음 중 온라인상에서 응용프로그램을 자유롭게 사고팔 수 있는 장터는?

1. Podcast

4. 안드로이드

앱스토어는 애플리케이션 스토어의 준말로, 모바일 애플리케이션을 자유롭게 사고팔 수 있는 온라인 장터이다.

### Q5. 다음 중 웨어러블 기기와 가장 거리가 먼 것은?

웨어러블 기기는 몸에 착용할 수 있는 형태의 컴퓨터를 의미한다. 스마트폰은 모바일 장치이지만 일반적으로 웨어러블 기기로 분류하지 않는다.

## 시험 직전 체크포인트

### 반드시 암기하기

- 모바일 = 이동성을 가진 정보통신 환경
- 모바일 장치 = 휴대폰, PDA, 스마트폰, 태블릿 PC
- PDA = Personal Digital Assistants
- 스마트폰 = 휴대폰 + PDA 기능 + 컴퓨터 기능
- 스마트폰 등장 핵심 = 2007년 애플 아이폰
- 모바일 시장 변화 = 음성 중심 → 모바일 서비스 중심
- 운영체제 = 하드웨어와 응용프로그램 사이의 중재 역할
- 모바일 운영체제 = iOS, Android, KaiOS, Windows Phone, Symbian, BlackBerry, LiMo
- BREW = 퀄컴, 휴대폰용 폐쇄형 플랫폼
- WIPI = 2005~2009년 국내 휴대폰 의무 탑재
- 안드로이드 = 구글, OHA, 리눅스 기반, 오픈소스, 자바 개발
- iOS = 애플, 직관적 UI, 앱스토어, 보안성
- KaiOS = 저사양·저전력 스마트 피처폰
- 앱스토어 = 모바일 콘텐츠 장터
- 카카오톡 = 무선이동통신 기반 무료 채팅 서비스
- Podcast = iPod + broadcast
- 웨어러블 기기 = 착용 가능한 컴퓨터
- 웨어러블 예 = 스마트워치, 스마트 안경, 의복형 기기
