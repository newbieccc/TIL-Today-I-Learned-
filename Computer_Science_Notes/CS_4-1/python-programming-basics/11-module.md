## 2. 모듈의 이해

### 2-1. 라이브러리와 프레임워크

- 파이썬은 다양한 라이브러리와 프레임워크를 통해 기능을 확장할 수 있음
- 라이브러리는 파이썬에 없는 추가 기능을 제공함
- 대표적으로 다음과 같은 분야에서 활용됨
    - 수학 계산
    - 데이터 분석
    - 그래프 시각화
    - 이미지 처리
    - 네트워크
    - 머신러닝 등

### 핵심 정리

- 파이썬은 기본 문법만으로 끝나는 언어가 아니라,
  필요한 기능을 `import`하여 확장하며 사용하는 언어임
- 라이브러리와 모듈을 이해하면 더 많은 기능을 손쉽게 사용할 수 있음

## 3. 모듈의 개념

### 3-1. 모듈이란?

- 모듈은 **함수, 상수 또는 클래스를 모아 놓은 집합체**임
- 하나의 단위로 묶어 재사용할 수 있게 만든 것임

### 구성 요소

| 구성 요소    | 의미                |
|----------|-------------------|
| 함수       | 특정 작업을 처리         |
| 상수 또는 변수 | 변하지 않는 값 또는 저장된 값 |
| 클래스      | 관련 데이터와 기능을 묶은 구조 |

### 핵심 정리

- 모듈은 관련 기능을 하나의 파일로 묶어 둔 것임
- 파이썬에서는 보통 `.py` 파일 하나가 모듈로 사용됨
- 모듈을 사용하면 코드 재사용성과 유지보수성이 좋아짐

## 4. 모듈, 패키지, 라이브러리

### 4-1. 모듈

- 클래스, 함수, 상수를 모아 놓은 `.py` 파일

### 4-2. 패키지

- 하위 패키지와 모듈을 모아 놓은 폴더
- 여러 모듈을 구조적으로 묶어 관리할 때 사용함

### 4-3. 라이브러리

- 패키지와 모듈을 모아 놓은 가장 큰 집합
- 특정 분야의 기능을 편리하게 사용하기 위해 제공됨

### 구조 정리

| 구분    | 의미                          |
|-------|-----------------------------|
| 모듈    | 함수, 상수, 클래스가 들어 있는 `.py` 파일 |
| 패키지   | 여러 모듈과 하위 패키지를 모은 폴더        |
| 라이브러리 | 여러 패키지와 모듈을 모은 큰 기능 집합      |

### 핵심 정리

- 모듈 < 패키지 < 라이브러리 순서로 범위가 커짐
- 실제 개발에서는 필요한 라이브러리나 모듈을 가져와서 사용함

## 5. 모듈의 등록

### 5-1. `import` 구문

### 구문 형식

    import 모듈이름 [as 별칭]

### 의미

- 파이썬 모듈을 프로그램 내부에서 사용할 수 있게
  네임스페이스에 추가하는 명령어임
- 모듈을 등록한 후에는 다음과 같은 방식으로 사용할 수 있음

  모듈이름.변수
  모듈이름.함수()
  모듈이름.클래스

### 예시

    import math

    print(math.sqrt(16))
    print(math.pi)

### 별칭 사용 예시

    import math as m

    print(m.sqrt(16))
    print(m.pi)

### 핵심 정리

- `import 모듈명`을 사용하면 `모듈명.멤버명` 형식으로 접근함
- `as`를 사용하면 긴 모듈 이름을 짧은 별칭으로 사용할 수 있음

---

### 5-2. `from ~ import` 구문

### 구문 형식

    from 모듈이름 import 메소드, 함수, 클래스

    from 모듈이름 import *

### 의미

- 특정 모듈에서 필요한 함수, 변수, 클래스만 가져와 사용할 수 있음
- 이 경우 모듈 이름 없이 바로 사용할 수 있음

### 예시

    from math import sqrt

    print(sqrt(16))

### 여러 함수 가져오기

    from math import sqrt, factorial

    print(sqrt(16))
    print(factorial(5))

### 전체 가져오기

    from math import *

    print(sqrt(16))
    print(pi)

### 핵심 정리

- `import math`
    - `math.sqrt()`처럼 모듈 이름을 붙여 사용
- `from math import sqrt`
    - `sqrt()`처럼 바로 사용 가능
- `from 모듈 import *`는 편리하지만,
  이름 충돌이 생길 수 있으므로 주의가 필요함

## 7. 네임스페이스

### 7-1. 네임스페이스의 이해

- 네임스페이스는 특정 객체를 이름에 따라 구분할 수 있는 범위임
- 변수, 함수, 클래스 등의 이름이 저장되고 관리되는 공간임

### 7-2. 네임스페이스의 종류

| 종류         | 의미                            |
|------------|-------------------------------|
| 지역 네임스페이스  | 함수 또는 메소드 내부의 이름 공간           |
| 전역 네임스페이스  | 모듈 전체에서 통용되는 이름 공간            |
| 빌트인 네임스페이스 | 모든 코드 범위에서 사용할 수 있는 파이썬 내장 영역 |

### 핵심 정리

- 같은 이름이라도 어느 네임스페이스에 속하느냐에 따라 의미가 달라질 수 있음
- 모듈을 `import`하면 해당 모듈 이름이 네임스페이스에 등록됨

## 8. 모듈 제거와 갱신

### 8-1. 모듈 제거

### 구문 형식

    del 등록된모듈이름

### 의미

- 네임스페이스 내 모듈의 멤버 식별자를 제거함

### 예시

    import math

    del math

---

### 8-2. 모듈의 갱신

- 파이썬은 효율성을 위해 불러온 모듈을 메모리에 저장함
- 이미 `import`한 모듈은 다시 `import`해도 변경 사항이 바로 반영되지 않을 수 있음
- 이때 `importlib.reload()`를 사용함

### 예시

    import importlib
    import mymodule

    importlib.reload(mymodule)

### 핵심 정리

- `del`은 네임스페이스에서 이름을 제거함
- 모듈을 수정한 뒤 다시 반영하려면 `importlib.reload()`를 사용할 수 있음

## 9-2. `random` 모듈

### 개념

- 난수(random number)는 특정한 순서나 규칙적 의미가 없는 임의의 수를 의미함
- `random` 모듈은 난수 관련 생성 기능을 제공함
- 특정 범위의 난수를 발생시킬 수 있음
- 난수 발생기는 시드(seed)를 기반으로 동작함

### 주요 멤버

| 멤버                   | 의미                                     |
|----------------------|----------------------------------------|
| `random()`           | 0~1 사이의 숫자 중 난수 발생                     |
| `randint(a, b)`      | `a`부터 `b` 사이의 숫자 중 난수 발생               |
| `randrange(a, b, c)` | `a`부터 `b` 사이에서 `c` 간격으로 나열된 숫자 중 난수 발생 |
| `choice(sequence)`   | 주어진 항목 중 하나를 랜덤하게 반환                   |
| `sample(sequence)`   | 랜덤하게 여러 개의 원소를 선택                      |
| `shuffle(sequence)`  | 시퀀스의 순서를 랜덤하게 섞음                       |

### 예시

    import random

    print(random.random())
    print(random.randint(1, 10))
    print(random.choice(["가위", "바위", "보"]))

### 핵심 정리

- `random` 모듈은 난수 생성이 필요한 프로그램에 사용됨
- 게임, 추첨, 확률 실험, 시뮬레이션 등에 활용 가능함

## 10. `random` 모듈 활용 예시

### 10-1. 가위바위보 게임 프로그램

### 기능

- 사용자가 가위, 바위, 보 중 하나를 입력함
- 컴퓨터도 랜덤으로 가위, 바위, 보 중 하나를 선택함
- 승패를 판단하여 출력함

### 예시 코드

    import random

    options = ["가위", "바위", "보"]

    user = input("가위, 바위, 보를 입력: ")
    com = random.choice(options)

    print("컴퓨터:", com)

    if user == com:
        print("비겼다!")
    elif user == "바위" and com == "가위":
        print("이겼다!")
    elif user == "보" and com == "바위":
        print("이겼다!")
    elif user == "가위" and com == "보":
        print("이겼다!")
    else:
        print("졌다!")

### 핵심 정리

- `random.choice()`를 사용하면 리스트에서 임의의 원소 하나를 선택할 수 있음
- 게임 프로그램에서 난수를 활용하면 매번 다른 결과를 만들 수 있음

### 10-2. 로또 번호 비교 예시

### 기능

- 사용자가 1~45 사이의 번호 6개를 입력함
- 컴퓨터가 1~45 사이에서 6개의 로또 번호를 랜덤하게 생성함
- 사용자가 맞힌 개수를 계산함

### 예시 코드

    import random

    guess_str = input("1~45 번호 6개를 입력하세요: ").split(" ")
    guess_list = [int(i) for i in guess_str if int(i) < 46]

    if len(guess_list) == 6:
        lotto_list = random.sample(range(1, 46), 6)
        print("당첨 숫자는", lotto_list, "입니다.")

        hit_count = 0

        for i in guess_list:
            for j in lotto_list:
                if i == j:
                    hit_count += 1

        print(str(hit_count) + "개 맞혔습니다.")
    else:
        print("45 이하의 번호 6개를 넣으세요!")

### 핵심 정리

- `random.sample(range(1, 46), 6)`은 1부터 45까지 중복 없이 6개를 뽑음
- `range(1, 46)`에서 46은 포함되지 않음

### 10-3. 스무고개 프로그램

### 기능

- 컴퓨터가 1~100 사이의 숫자를 하나 정함
- 사용자는 20번의 기회 안에 숫자를 맞힘
- 입력값이 정답보다 크거나 작은지 알려 줌

### 예시 코드

    import random

    hit_number = random.randint(1, 101)
    print(hit_number)

    guess_count_list = range(1, 21)
    passed = False

    for guess_count in guess_count_list:
        guess = int(input("숫자를 맞혀보세요! "
                          + str(guess_count) + "번째 시도: "))

        if hit_number == guess:
            passed = True
            break
        elif hit_number > guess:
            print("더 큰 수입니다.")
        else:
            print("더 작은 수입니다.")

    if passed == True:
        print("맞혔습니다. 축하합니다.")
    else:
        print("모든 기회를 다 사용하셨습니다.")

### 핵심 정리

- `randint()`는 지정한 범위의 난수를 생성함
- 반복문과 조건문을 함께 사용하여 게임 흐름을 만들 수 있음

## 9-3. `time` 모듈

### 개념

- `time` 모듈은 에포크 시간을 얻어 다양한 형식으로 표시하는 기능을 제공함
- 시간 측정, 프로그램 실행 지연, 시간 형식 변환 등을 처리함
- 유닉스 에포크는 `1970-01-01 00:00:00 GMT`를 기준으로 함
- 현재 시간은 에포크 기준으로 경과한 시간으로 표현할 수 있음

### 주요 멤버

| 멤버                    | 의미                              |
|-----------------------|---------------------------------|
| `time()`              | 1970.1.1 자정 이후 누적된 초를 실수 단위로 반환 |
| `gmtime()`            | `struct_time` 형식으로 기준 시각 변환     |
| `localtime(time)`     | 입력된 초를 변환하여 지방표준시 기준 시각으로 변환    |
| `strftime(str, time)` | 지정된 형식으로 시각을 변환                 |
| `ctime()`             | 현재 시간을 반환                       |
| `sleep(n)`            | 현재 동작 중인 프로세스를 주어진 `n`초만큼 멈춤    |

### 예시

    import time

    print(time.time())
    print(time.ctime())

    time.sleep(3)

### 핵심 정리

- `time.time()`은 현재 시간을 초 단위로 반환함
- `time.sleep(n)`은 코드 실행을 `n`초 동안 멈춤
- 실행 시간 측정이나 지연 처리에 사용할 수 있음

## 11. `time` 모듈 활용 예시

### 11-1. 소수 찾기 프로그램

### 기능

- 1부터 50000 사이의 소수를 찾고 실행 시간을 출력함

### 처리 흐름

1. 소수 판별 함수 `isprime()` 작성
2. 시작 시간 저장
3. 2부터 50000까지 반복
4. 소수이면 출력하고 개수 증가
5. 종료 시간 저장
6. 실행 시간 출력

### 예시 코드

    import time

    max_num = 50000

    def isprime(x):
        for i in range(2, x):
            if x % i == 0:
                return False
        return True

    pn_count = 0
    start_time = time.time()

    for i in range(2, max_num + 1):
        if isprime(i):
            pn_count += 1
            print(i, end=" ")

            if pn_count % 5 == 0:
                print("")

    end_time = time.time()

    print(format(end_time - start_time, ".3f"), "초 실행했습니다.")

### 코드 해석

- `time.time()`
    - 현재 시간을 초 단위로 반환함
- `start_time`
    - 프로그램 시작 시각 저장
- `end_time`
    - 프로그램 종료 시각 저장
- `end_time - start_time`
    - 실행 시간 계산
- `range(2, max_num + 1)`
    - 2부터 `max_num`까지 반복
    - `range(a, b)`에서 `a`는 포함되고, `b`는 포함되지 않음

### 핵심 정리

- `time` 모듈은 코드 실행 시간을 측정할 때 유용함
- 소수 판별 함수와 반복문을 결합하여 성능 측정 프로그램을 만들 수 있음

## 12. 정리하기

1. 모듈은 함수, 상수, 클래스를 모아 놓은 파이썬 파일임
    - 패키지는 이러한 모듈들의 폴더 묶음
    - 라이브러리는 패키지와 모듈을 모아 놓은 가장 큰 집합체

2. 네임스페이스는 변수와 함수 등의 이름이 구분되고 관리되는 공간임
    - 이름은 지역, 전역, 빌트인 네임스페이스 순으로 이루어짐

3. 외부 모듈을 사용하기 위해서는 `import` 구문을 사용해 네임스페이스에 추가해야 함

5. `random` 모듈은 임의의 숫자를 생성하는 난수 발생 기능을 제공함
    - `randint()`, `choice()`, `shuffle()` 등을 활용하여 게임이나 추첨 프로그램 구현에 활용됨

6. `time` 모듈은 유닉스 에포크를 기준으로 경과 시간을 측정하거나,
   현재 시각 확인, 프로세스 지연 `sleep()` 기능 등을 제공함

### 13-5. 주요 모듈

## 14. 연습문제 정리

### Q1. 파이썬에서 관련된 함수, 변수, 클래스 등을 하나의 파일로 묶어 재사용성과 유지보수성을 높이는 기본 단위는 무엇인가?

1. 클래스
2. 모듈
3. 인터페이스
4. 네임스페이스

- 정답: **2**

### 해설

- 관련 함수, 변수, 클래스 등을 하나의 `.py` 파일로 묶은 것이 모듈임
- 모듈은 코드 재사용성과 유지보수성을 높이기 위한 기본 단위임

### Q2. 특정 모듈 `math`에서 특정 함수 `sqrt`만을 가져오기 위한 올바른 구문 형식은?

1. `import math.sqrt`
2. `import sqrt from math`
3. `from math import sqrt`
4. `math import sqrt`

- 정답: **3**

### 해설

- 특정 모듈에서 특정 함수만 가져올 때는 다음 형식을 사용함

  from math import sqrt

- 이 경우 `math.sqrt()`가 아니라 `sqrt()`처럼 바로 사용할 수 있음

---

### Q3. 코드의 실행을 잠시 3초 동안 멈추고자 할 때 사용하는 `time` 모듈의 함수는?

1. `time.stop(3)`
2. `time.pause(3)`
3. `time.wait(3)`
4. `time.sleep(3)`

- 정답: **4**

### 해설

- `sleep(n)`은 현재 동작 중인 프로세스를 주어진 `n`초만큼 멈추는 함수임
- 따라서 3초 동안 멈추려면 다음과 같이 작성함

  time.sleep(3)

## 15. 시험 직전 체크포인트

- 모듈은 함수, 상수, 클래스를 모아 놓은 `.py` 파일이다.
- 패키지는 여러 모듈을 모아 놓은 폴더이다.
- 라이브러리는 패키지와 모듈을 모아 놓은 큰 집합이다.
- `import 모듈명`을 사용하면 `모듈명.멤버명` 형식으로 접근한다.
- `as`를 사용하면 모듈에 별칭을 붙일 수 있다.
- `from 모듈 import 함수`를 사용하면 모듈명 없이 바로 함수 사용이 가능하다.
- `dir()`은 네임스페이스에 등록된 이름 목록을 확인한다.
- `help()`는 함수나 메소드의 사용법을 확인한다.
- 네임스페이스는 이름이 저장되고 관리되는 공간이다.
- `del 모듈명`은 네임스페이스에서 모듈 이름을 제거한다.
- `importlib.reload(모듈명)`은 수정된 모듈을 다시 불러올 때 사용한다.
- `math` 모듈은 수학 함수와 상수를 제공한다.
- `random` 모듈은 난수 생성 기능을 제공한다.
- `random.choice()`는 시퀀스에서 임의의 항목 하나를 선택한다.
- `random.sample()`은 여러 개의 원소를 무작위로 선택한다.
- `time.time()`은 현재 시간을 초 단위로 반환한다.
- `time.sleep(n)`은 실행을 `n`초 동안 멈춘다.
- `range(a, b)`는 `a`는 포함하고 `b`는 포함하지 않는다.
