## 2. 함수의 이해

### 2-1. 함수의 개념

- 함수는 **특정 작업을 수행하는 명령문의 집합**임
- 특정 작업을 함수의 이름으로 대체할 수 있음
- 유사한 유형의 문제를 해결할 수 있도록 설계한 추상화 단위임

### 2-2. 정의 주체에 따른 분류

| 구분        | 설명                    |
|-----------|-----------------------|
| 내장 함수     | 파이썬 내부에 이미 만들어져 있는 함수 |
| 사용자 정의 함수 | 사용자의 목적에 따라 직접 정의한 함수 |

### 2-3. 반환값 유무에 따른 분류

| 구분         | 설명                | 예                             |
|------------|-------------------|-------------------------------|
| 반환값이 없는 함수 | 내부 동작 수행 후 종료     | `print()`                     |
| 반환값이 있는 함수 | 내부 동작 수행 후 결과를 반환 | `input()`, `max()`, `int()` 등 |

### 핵심 정리

- 함수는 반복적으로 사용되는 특정 작업을 이름으로 묶어 놓은 것임
- 함수는 코드의 재사용성과 가독성을 높여 줌
- 함수는 내장 함수와 사용자 정의 함수로 나눌 수 있음
- 함수는 반환값이 있는지 없는지에 따라 구분할 수 있음

---

## 3. 반환값이 없는 함수

### 3-1. 반환값이 없는 함수 정의

- 반환값이 없는 함수는 내부 동작을 수행한 뒤 결과값을 반환하지 않고 종료함

### 구문 형식

    def 함수이름(매개변수 리스트):
        명령 블록

### 3-2. 구성 요소

- `def`
    - 함수를 정의할 때 사용하는 키워드
- `함수이름`
    - 함수를 구분하기 위한 식별자
- `매개변수(parameter) 리스트`
    - 함수 내부로 전달할 값을 받는 변수 목록
    - 0개 이상의 값을 전달할 수 있음
- `명령 블록`
    - 함수가 호출되었을 때 실행할 명령문 집합

### 3-3. 매개변수의 특징

- 매개변수 리스트는 0개 이상의 값을 함수 내부로 전달함
- 함수 내부에서 매개변수 리스트에 포함된 각 매개변수는 변수와 동일하게 사용됨

### 핵심 정리

- 함수는 `def 함수이름(...):` 형식으로 정의함
- 함수 안의 명령문은 반드시 들여쓰기 해야 함
- 매개변수는 함수 외부에서 전달된 값을 함수 내부에서 사용할 수 있게 해 줌

## 4. 함수의 호출과 실행 흐름

### 4-1. 원뿔 부피 계산 함수 예시

    # 원뿔 부피 계산 함수
    def prt_cone_vol(r, h):
        if r > 0 and h > 0:
            # r, h 모두 양수일 때
            vol = 1 / 3 * 3.14 * r ** 2 * h
            print("원뿔의 부피는", vol, "입니다.")
        else:
            # r, h가 음수일 때
            print("반지름과 높이 값에 양수를 입력하세요")

    # 반지름 30, 높이 50
    rad = 30
    hei = 50

    prt_cone_vol(rad, hei)

### 4-2. 실행 흐름

- 함수 정의 구간
    - `def prt_cone_vol(r, h):` 아래에 함수 내용을 작성함
    - 이 시점에는 함수가 실행되는 것이 아니라, 함수가 정의됨
- 프로그램 실행 구간
    - `rad = 30`, `hei = 50`을 저장함
    - `prt_cone_vol(rad, hei)`를 호출하면 함수 내부 명령문이 실행됨

### 4-3. 코드 해석

- `prt_cone_vol(r, h)`
    - 반지름 `r`, 높이 `h`를 매개변수로 받는 함수
- `if r > 0 and h > 0`
    - 반지름과 높이가 모두 양수인지 검사함
- `vol = 1 / 3 * 3.14 * r ** 2 * h`
    - 원뿔 부피를 계산함
- `print(...)`
    - 계산 결과 또는 오류 안내 문구를 출력함
- `prt_cone_vol(rad, hei)`
    - `rad`, `hei` 값을 함수에 전달하여 실행함

### 핵심 정리

- 함수는 정의만 해서는 실행되지 않음
- 함수 이름과 인수를 사용해 호출해야 실행됨
- 함수 호출 시 전달한 값은 매개변수에 대응되어 함수 내부에서 사용됨

## 5. 반환값이 있는 함수

### 5-1. 반환값이 있는 함수 정의

- 반환값이 있는 함수는 실행 후 결과값을 남기는 함수임
- `return` 키워드를 사용하여 결과값을 함수 호출 위치로 전달함

### 구문 형식

    def 함수이름(매개변수 리스트):
        명령 블록
        return 반환값 리스트

### 5-2. `return`의 역할

- 함수 내부의 실행 결과를 호출한 곳으로 전달함
- 함수 내부에서 여러 개의 `return`을 사용할 수 있음
- `return`이 실행되면 함수는 즉시 종료됨

### 핵심 정리

- 반환값이 필요한 경우 `return`을 사용함
- `return`은 결과값 전달과 함수 종료 역할을 함께 수행함

---

## 6. 반환값이 있는 함수 예시

### 6-1. 원뿔 부피 반환 함수

    # 원뿔 부피 계산 함수 정의
    def rtn_cone_vol(r, h):
        if r > 0 and h > 0:
            # r, h 모두 양수일 때
            return 1 / 3 * 3.14 * r ** 2 * h
        else:
            # r, h가 음수일 때
            return -1

    # 반지름 10, 높이 50인 원뿔 부피 계산
    rad = 10
    hei = 50

    print(format(rtn_cone_vol(rad, hei), ">10.3f"))

### 6-2. 코드 해석

- `rtn_cone_vol(r, h)`
    - 원뿔의 부피를 계산하여 반환하는 함수
- `return 1 / 3 * 3.14 * r ** 2 * h`
    - 계산 결과를 함수 호출 위치로 반환함
- `return -1`
    - 잘못된 입력값이 들어온 경우 오류를 의미하는 값으로 `-1` 반환
- `format(..., ">10.3f")`
    - 반환된 실수 값을 지정된 형식으로 출력함

### 핵심 정리

- 반환값이 있는 함수는 계산 결과를 다른 코드에서 다시 활용할 수 있음
- `print()`는 출력만 하지만, `return`은 값을 되돌려 줌

## 7. 실수 데이터 형식화

### 7-1. `format()` 함수

- `format()` 함수는 데이터를 지정한 형식에 맞게 변환할 때 사용함
- 실수 데이터를 원하는 자릿수와 정렬 방식으로 표현할 수 있음

### 7-2. 형식 지정자 예시

    format(3.141592, ">10.3f")

### 7-3. 형식 지정자의 의미

| 구성   | 의미          |
|------|-------------|
| `>`  | 오른쪽 정렬      |
| `10` | 전체 출력 폭 10칸 |
| `.3` | 소수점 이하 3자리  |
| `f`  | 실수 데이터 타입   |

### 핵심 정리

- `format()`은 출력 형식을 정리할 때 유용함
- `">10.3f"`는 전체 10칸 폭에서 오른쪽 정렬하고 소수점 이하 3자리까지 표현함

## 8. 동시 할당

### 8-1. 동시 할당의 개념

- 복수 개의 변수에 값을 동시에 할당하는 방법임
- 변수의 개수에 상응하는 값을 콤마(`,`)로 나열함

### 8-2. 예시

    temp = 27
    season = "summer"

    temp, season = 27, "summer"

### 코드 해석

- `temp`에는 `27`이 저장됨
- `season`에는 `"summer"`가 저장됨
- 여러 변수와 값을 한 줄에서 동시에 처리할 수 있음

### 핵심 정리

- 동시 할당은 여러 변수에 값을 한 번에 저장할 수 있는 방식임
- 변수 개수와 값 개수가 맞아야 함

---

## 9. 교환(swap)

### 9-1. 교환의 개념

- 복수의 변수에 할당된 값을 맞바꾸는 연산임

### 9-2. 일반적인 교환 방식

    temp = rad
    rad = hei
    hei = temp

### 9-3. 파이썬의 동시 할당을 이용한 교환

    rad, hei = hei, rad

### 코드 해석

- 일반적인 방식에서는 임시 변수 `temp`가 필요함
- 파이썬에서는 동시 할당을 이용해 두 변수 값을 간단히 교환할 수 있음

### 핵심 정리

- 파이썬에서는 `a, b = b, a` 형식으로 두 변수의 값을 쉽게 교환할 수 있음

## 10. 함수의 실행과 변수 값

### 10-1. 함수 내부 변수 변경 예시

    # 원뿔 부피 계산 함수 정의
    def rtn_cone_vol(r, h):
        if r > 0 and h > 0:
            # r, h를 0으로 초기화
            r, h = 0, 0
            return 1 / 3 * 3.14 * r ** 2 * h
        else:
            return -1

    # 반지름 10, 높이 50
    rad = 10
    hei = 50

    format(rtn_cone_vol(rad, hei), ">10.3f")
    print(rad, hei)

### 10-2. 코드 해석

- `r`, `h`는 함수 내부의 매개변수임
- 함수 안에서 `r`, `h` 값을 변경해도
  함수 밖의 `rad`, `hei` 값이 직접 바뀌지는 않음
- 함수 밖의 변수와 함수 내부의 매개변수는 서로 다른 영역에서 관리됨

### 핵심 정리

- 함수 내부에서 매개변수 값을 바꾸더라도,
  함수 밖의 변수 값이 항상 같이 바뀌는 것은 아님
- 이 개념은 변수의 스코프와 연결됨

---

## 11. 변수의 스코프(scope)

### 11-1. 스코프의 개념

- 스코프는 프로그램에서 변수가 참조될 수 있는 영역임

### 11-2. 전역변수와 지역변수

| 구분   | 설명                              |
|------|---------------------------------|
| 전역변수 | 함수 외부에서 선언되며 프로그램 전체 영역에서 접근 가능 |
| 지역변수 | 함수 내부에서 선언되며 선언된 함수 내부에서만 접근 가능 |

### 11-3. 예시

    x = 1

    def inc():
        y = x + 1
        print("y의 값은", y)

    inc()

    print("x의 값은", x)
    print("y의 값은", y)

### 11-4. 코드 해석

- `x`
    - 함수 외부에서 선언된 전역변수
- `y`
    - 함수 내부에서 선언된 지역변수
- 함수 내부에서는 `y`를 사용할 수 있음
- 함수 밖에서 `y`를 직접 출력하려 하면 오류가 발생할 수 있음

### 핵심 정리

- 전역변수는 함수 밖에서도 접근 가능함
- 지역변수는 함수 내부에서만 접근 가능함
- 함수 내부와 외부의 변수 영역을 구분해야 함

---

## 12. 기본 매개변수

### 12-1. 기본 매개변수의 개념

- 함수 호출 시 매개변수가 입력되지 않은 경우,
  함수 정의 시 지정된 기본값이 전달되는 매개변수임

### 12-2. 예시: `print()`의 `sep`

    print("Hello", "I love Python")

    print("Hello", "I love Python", sep=" ")

### 코드 해석

- `print()` 함수는 여러 데이터를 출력할 때 기본적으로 공백을 사이에 넣음
- `sep=" "`는 별도로 지정하지 않아도 기본값으로 적용되는 매개변수임

### 12-3. 기본 매개변수 정의 구문

    def 함수이름(매개변수 리스트, 매개변수=값리스트):
        명령 블록
        return 반환값리스트

### 12-4. 주의점

- 기본 매개변수는 일반 매개변수 앞에 위치할 수 없음

### 잘못된 예시

    def hello(msg="반갑습니다", name):
        print(name, msg)

### 올바른 예시

    def hello(name, msg="반갑습니다"):
        print(name, msg)

### 핵심 정리

- 기본 매개변수는 인수를 생략했을 때 사용할 기본값을 미리 지정하는 방법임
- 기본 매개변수는 일반 매개변수 뒤에 작성해야 함

---

## 13. 가변 매개변수

### 13-1. 가변 매개변수의 개념

- 함수에 전달되는 인수의 개수가 정해져 있지 않을 때 사용하는 매개변수임
- `*`를 붙여 표현함

### 13-2. 구문 형식

    def 함수이름(매개변수 리스트, *가변 매개변수):
        명령 블록
        return 반환값 리스트

### 13-3. 주의점

- 가변 매개변수는 일반 매개변수 앞에 위치할 수 없음
- 가변 매개변수는 함수 정의에서 1개만 사용 가능함

### 예시

    def total_sum(*numbers):
        result = 0

        for n in numbers:
            result += n

        return result

    print(total_sum(1, 2, 3))
    print(total_sum(1, 2, 3, 4, 5))

### 핵심 정리

- 가변 매개변수는 인수 개수가 일정하지 않을 때 사용함
- `*args` 형태로 많이 사용함
- 여러 값을 하나의 묶음으로 받아 처리할 수 있음

## 14. 핵심 요약

### 14-1. 함수

- 함수는 특정 작업을 수행하는 명령문의 집합임
- 특정 작업을 함수 이름으로 대체할 수 있음
- 내장 함수와 사용자 정의 함수로 나눌 수 있음

### 14-2. 반환값 유무

| 구분         | 설명               | 예                              |
|------------|------------------|--------------------------------|
| 반환값이 없는 함수 | 내부 동작 수행 후 종료    | `print()`                      |
| 반환값이 있는 함수 | 내부 동작 수행 후 결과 반환 | `input()`, `format()`, `int()` |

### 14-3. 함수 정의 형식

    def 함수이름(매개변수 리스트):
        명령 블록

### 14-4. 반환값 있는 함수 정의 형식

    def 함수이름(매개변수 리스트):
        명령 블록
        return 반환값 리스트

### 14-5. 꼭 기억할 개념

- `def`
    - 함수 정의 키워드

    - 함수 내부로 전달되는 값

    - 결과값을 반환하고 함수 실행을 종료함

    - 데이터 형식화 함수
- 동시 할당
    - 여러 변수에 값을 한 번에 저장함
- 교환
    - `a, b = b, a`
- 스코프
    - 변수가 참조될 수 있는 영역
- 기본 매개변수
    - 인수를 생략했을 때 사용할 기본값
- 가변 매개변수
    - 개수가 정해지지 않은 여러 인수를 받는 매개변수

## 15. 연습문제 정리

### Q1. 다음 중 반환값에 따른 함수의 종류를 구분할 때, 나머지와 다른 하나는?

1. `print`
2. `input`
3. `format`
4. `int`

- 정답: **1**

### 해설

- `print()`는 값을 화면에 출력하는 함수이며,
  일반적으로 반환값을 활용하는 함수로 분류하지 않음
- `input()`, `format()`, `int()`는 실행 후 결과값을 반환하여 다른 코드에서 활용할 수 있음

### Q2. 다음 코드의 실행 결과는?

    temp, season = 27, "summer"
    season, temp = temp, season
    print(season)

1. `"summer"`
2. `27`
3. `season`
4. `print`

- 정답: **2**

### 해설

- 처음에는 다음과 같이 저장됨

  temp = 27
  season = "summer"

- 이후 동시 할당으로 두 값이 교환됨

  season, temp = temp, season

- 따라서 `season`에는 `27`이 저장됨
- 출력 결과는 다음과 같음

  27

### Q3. 다음 코드의 실행 결과는?

    x = 1

    def update():
        x = 2
        x = x + 1

    update()
    print(x)

1. `1`
2. `2`
3. `3`
4. 오류발생

- 정답: **1**

### 해설

- 함수 내부의 `x`는 지역변수임
- 함수 밖의 `x = 1`은 전역변수임
- `update()` 함수 안에서 `x` 값을 변경해도 전역변수 `x`는 바뀌지 않음
- 따라서 `print(x)`는 함수 밖의 전역변수 값인 `1`을 출력함

## 16. 시험 직전 체크포인트

- 함수는 특정 작업을 수행하는 명령문의 집합이다.
- 함수는 `def 함수이름(매개변수):` 형식으로 정의한다.
- 함수는 정의만으로 실행되지 않고 호출해야 실행된다.
- 매개변수는 함수 내부로 값을 전달하는 통로이다.
- 반환값이 없는 함수는 내부 동작 수행 후 종료한다.
- 반환값이 있는 함수는 `return`을 통해 값을 돌려준다.
- `return`이 실행되면 함수는 즉시 종료된다.
- `format(값, 형식지정자)`는 출력 형식을 지정한다.
- 동시 할당은 여러 변수에 값을 한 번에 할당하는 방식이다.
- 파이썬에서는 `a, b = b, a`로 값을 쉽게 교환할 수 있다.
- 전역변수는 함수 밖에서 선언되어 프로그램 전체에서 접근 가능하다.
- 지역변수는 함수 내부에서 선언되어 함수 내부에서만 접근 가능하다.
- 기본 매개변수는 일반 매개변수 뒤에 작성해야 한다.
- 가변 매개변수는 `*`를 붙여 사용하며, 하나만 사용할 수 있다.
