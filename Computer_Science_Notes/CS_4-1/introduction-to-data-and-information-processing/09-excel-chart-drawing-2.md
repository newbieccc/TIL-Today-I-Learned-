## 1. 히스토그램

### 1.1 도수분포표

도수분포표(frequency table)는 겹치지 않는 몇 개의 범주, 즉 계급에 속한 관측치의 개수를 요약하여 작성한 표이다.

도수분포표는 정성적 데이터뿐만 아니라 정량적 데이터에도 적용할 수 있다.

| 구분    | 설명                     |
|-------|------------------------|
| 도수분포표 | 계급별 관측치 개수를 요약한 표      |
| 계급    | 자료를 나누는 구간 또는 범주       |
| 도수    | 각 계급에 속하는 자료의 개수       |
| 적용 자료 | 정성적 데이터, 정량적 데이터 모두 가능 |

정량적 데이터에 대한 도수분포표를 작성할 때는 계급 정의에 세심한 고려가 필요하다.

---

### 1.2 도수분포표 작성 방법

도수분포표 작성 절차는 다음과 같다.

1. 데이터 수를 센다.
2. 데이터의 최대값과 최소값을 구한다.
3. 계급의 수를 결정한다.
4. 계급의 폭을 결정한다.
5. 계급의 경계값과 중앙값을 구한다.
6. 각 계급에 속하는 도수를 세어 도수분포표를 작성한다.

---

### 1.3 계급 수 결정

계급의 수는 자료의 개수 n을 기준으로 대략 다음과 같이 정한다.

    계급의 수 ≈ √n ± 2~3

보통 계급 수는 5개에서 20개 정도로 정한다.

계급 수가 너무 적으면 분포의 특징이 지나치게 단순해지고, 계급 수가 너무 많으면 분포의 전체적인 흐름을 보기 어렵다.

---

### 1.4 계급의 폭

계급의 폭은 다음과 같이 구한다.

    계급의 폭 = (최대값 - 최소값) / 계급의 수

계급의 폭을 정할 때는 자료가 너무 한쪽에 몰리거나 너무 세분되지 않도록 적절하게 조정해야 한다.

---

### 1.5 계급 경계값과 계급값

첫 번째 계급의 경계값과 계급값은 다음과 같이 구한다.

    첫 번째 하한경계값 = 최소값 - (측정의 최소단위 / 2)

    첫 번째 상한경계값 = 하한경계값 + 계급의 폭

    계급값 = (하한경계값 + 상한경계값) / 2

| 용어    | 의미              |
|-------|-----------------|
| 하한경계값 | 계급의 아래쪽 경계      |
| 상한경계값 | 계급의 위쪽 경계       |
| 계급값   | 해당 계급을 대표하는 중앙값 |

## 2. 히스토그램

### 2.1 히스토그램의 의미

히스토그램(histogram)은 연속형 데이터에 대해 데이터의 특징이나 분포 모양을 살펴볼 때 널리 이용되는 그래프이다.

히스토그램은 수평축 위에 계급구간을 표시하고, 그 위로 각 계급의 상대도수에 비례하는 넓이의 직사각형을 그린 것이다.

| 구분    | 설명                   |
|-------|----------------------|
| 대상 자료 | 연속형 데이터              |
| 목적    | 데이터의 특징과 분포 모양 파악    |
| 수평축   | 계급구간                 |
| 직사각형  | 각 계급의 도수 또는 상대도수를 표현 |

---

### 2.2 히스토그램으로 확인할 수 있는 것

히스토그램을 통해 다음을 확인할 수 있다.

- 데이터가 어느 구간에 많이 몰려 있는지
- 분포가 대칭인지 비대칭인지
- 봉우리가 하나인지 여러 개인지
- 특이값이 있는지
- 전체적인 분포 형태가 어떤지

---

### 2.3 엑셀에서 히스토그램 그리기 절차

엑셀에서 데이터 분석 도구를 이용하여 히스토그램을 그리는 절차는 다음과 같다.

1. [데이터] 탭을 선택한다.
2. [분석] 그룹의 [데이터 분석] 명령을 선택한다.
3. 분석 도구 중 `히스토그램`을 선택한다.
4. [확인]을 누른다.
5. 대화상자에서 `입력 범위`에 데이터 범위를 지정한다.
6. `출력 옵션`에서 히스토그램을 출력할 셀 주소를 입력하거나 새로운 워크시트를 선택한다.
7. `차트 출력`을 클릭한다.
8. 결과를 확인한다.

---

### 2.4 데이터 분석 명령이 없는 경우

[데이터] 탭에서 [데이터 분석] 명령이 보이지 않으면 분석 도구를 추가해야 한다.

절차:

1. [파일] 단추 클릭
2. [옵션] 선택
3. [추가 기능] 선택
4. [분석 도구] 항목 선택
5. 확인 후 [데이터 분석] 명령이 나타나는지 확인

---

### 2.5 히스토그램 수정

엑셀에서 생성된 히스토그램의 막대 간격을 조정할 수 있다.

절차:

1. 히스토그램의 여러 막대 중 하나를 선택한다.
2. 마우스 오른쪽 단추를 클릭한다.
3. 단축메뉴에서 [데이터 계열 서식]을 선택한다.
4. 데이터 계열 서식 대화상자에서 [계열 옵션] 탭을 선택한다.
5. `간격 너비` 입력란에 150 대신 0을 입력한다.
6. [확인]을 선택한다.

간격 너비를 0으로 설정하면 히스토그램 막대가 붙어서 표시되어 연속형 자료의 분포를 더 자연스럽게 보여줄 수 있다.

---

### 2.6 차트 도구모음에서 히스토그램 그리기

엑셀에서는 데이터 분석 도구뿐 아니라 차트 도구모음을 이용해서도 히스토그램을 작성할 수 있다.

다만 시험에서는 다음 두 가지를 구분해서 기억한다.

| 방법           | 핵심                         |
|--------------|----------------------------|
| 데이터 분석 도구 이용 | [데이터] - [데이터 분석] - [히스토그램] |
| 차트 도구 이용     | [삽입] - [차트]에서 히스토그램 선택     |

## 3. 산점도와 상관분석

## 3.1 산점도

### 3.1.1 산점도의 의미

산점도(scatter plot)는 두 연속형 변수 사이의 관계를 그래프를 통해 살펴보기 위해 사용하는 그래프이다.

두 변수 사이의 연관성 정도를 살펴보기 위한 분석의 첫 단계로 사용된다.

| 구분    | 설명             |
|-------|----------------|
| 대상 자료 | 두 개의 연속형 변수    |
| 목적    | 두 변수 사이의 관계 확인 |
| 활용    | 상관분석 전 시각적 확인  |
| 차트 유형 | 분산형            |

---

### 3.1.2 산점도로 확인할 수 있는 것

산점도를 보면 다음을 확인할 수 있다.

- 두 변수 사이에 관계가 있는지
- 관계가 양의 방향인지 음의 방향인지
- 관계가 강한지 약한지
- 이상치가 있는지
- 직선적 관계인지 곡선적 관계인지

---

### 3.1.3 산점도 그리기 절차

엑셀에서 산점도를 그리는 절차는 다음과 같다.

1. 데이터를 선택한다.
2. [삽입] 리본 메뉴를 선택한다.
3. [차트] 그룹에서 `분산형`을 선택한다.
4. 분산형 차트 유형 중에서 적합한 세부 차트 모양을 선택한다.
5. 차트마법사 대화상자에서 계열 위치를 지정한다.
6. 출력 결과를 확인한다.

---

### 3.1.4 산점도 해석 예

강의 예시에서는 차량 무게와 연비 사이의 산점도를 제시한다.

산점도 결과를 보면 차량 무게가 커질수록 연비가 낮아지는 경향이 나타난다.

즉, 두 변수 사이에는 음의 관계가 있다.

| 변수 1  | 변수 2 | 해석                       |
|-------|------|--------------------------|
| 차량 무게 | 연비   | 차량 무게가 증가할수록 연비가 감소하는 경향 |

---

## 4. 상관분석

### 4.1 상관분석의 의미

상관분석은 두 변수 사이의 선형관계, 즉 직선관계의 정도를 알아보는 분석이다.

상관계수는 두 변수 간 선형관계의 정도를 나타내는 측도이다.

---

### 4.2 상관계수

상관계수는 일반적으로 ρ로 표현하며, -1과 1 사이의 값을 가진다.

    -1 ≤ ρ ≤ 1

| 상관계수        | 해석         |
|-------------|------------|
| ρ > 0       | 양의 선형관계    |
| ρ < 0       | 음의 선형관계    |
| ρ = 0에 가까움  | 선형관계가 약함   |
| ρ = 1에 가까움  | 강한 양의 선형관계 |
| ρ = -1에 가까움 | 강한 음의 선형관계 |

---

### 4.3 상관계수의 절댓값

상관계수의 방향은 부호로 보고, 관계의 강도는 절댓값으로 판단한다.

| 기준 | 해석 |
|----|----|
|    | ρ  |가 1에 가까움 | 두 변수는 강한 선형 연관성 |
|    | ρ  |가 0에 가까움 | 두 변수는 약한 선형 연관성 |

예를 들어 ρ = -0.82라면 음의 상관관계이며, 절댓값이 1에 가까우므로 비교적 강한 선형관계라고 볼 수 있다.

---

### 4.4 산점도와 상관계수

산점도와 상관계수를 함께 보면 두 변수의 관계를 더 잘 이해할 수 있다.

| 산점도 모양          | 상관계수 해석               |
|-----------------|-----------------------|
| 오른쪽 위로 올라가는 형태  | 양의 상관                 |
| 오른쪽 아래로 내려가는 형태 | 음의 상관                 |
| 점들이 직선 주변에 밀집   | 강한 상관                 |
| 점들이 넓게 흩어짐      | 약한 상관                 |
| 곡선 형태           | 상관계수만으로 설명하기 어려울 수 있음 |

상관계수는 선형관계의 정도를 나타내므로, 곡선 형태의 관계는 상관계수가 낮게 나올 수 있다.

---

## 5. 엑셀에서 상관분석하기

### 5.1 상관분석 절차

엑셀에서 상관분석을 수행하는 절차는 다음과 같다.

1. [데이터] 탭을 선택한다.
2. [분석] 그룹에서 [데이터 분석] 명령을 선택한다.
3. 통계 데이터 분석 대화상자에서 `상관분석` 항목을 선택한다.
4. 대화상자에서 `입력범위`를 지정한다.
5. `데이터 방향`을 지정한다.
6. 첫째 행 정보를 변수 이름으로 사용하려면 `첫째 행 이름표 사용`을 체크한다.
7. 출력 옵션을 지정한다.
8. [확인]을 누른다.
9. 상관계수 결과를 확인한다.

---

### 5.2 상관분석 결과 해석 예

강의 예시에서 두 변수 간의 상관계수는 다음과 같다.

    r = -0.82

해석:

- 음의 상관관계이다.
- 절댓값이 1에 비교적 가까우므로 뚜렷한 선형관계가 있다.
- 차량 무게가 증가할수록 연비가 감소하는 경향이 있다고 볼 수 있다.

## 6. 실습: 성적데이터

성적데이터 실습에서는 다음 작업을 수행한다.

1. 히스토그램 작성: 통계분석 도구 이용
2. 히스토그램 작성: 차트 이용
3. 산점도 그리기
4. 상관계수 구하기

---

### 6.1 히스토그램 실습

성적데이터의 점수 분포를 보기 위해 히스토그램을 작성할 수 있다.

| 방법         | 설명                    |
|------------|-----------------------|
| 통계분석 도구 이용 | [데이터 분석] - [히스토그램]    |
| 차트 이용      | [삽입] - [차트] - [히스토그램] |

---

### 6.2 산점도 실습

두 과목 점수 사이의 관계를 확인하려면 산점도를 그린다.

예:

- 통계학 점수와 수학 점수의 관계
- 한 과목 점수가 높을수록 다른 과목도 높은지 확인
- 관계가 약한지 강한지 확인

---

### 6.3 상관계수 실습

산점도에서 두 변수 사이의 경향을 시각적으로 확인한 뒤, 상관분석을 통해 상관계수를 구한다.

상관계수로 두 변수 사이의 선형관계 정도를 수치로 확인할 수 있다.

---

## 7. 방사형 차트

### 7.1 방사형 차트의 활용

강의 연습문제에서는 음식점 3곳을 맛, 가격, 청결도, 서비스, 분위기 5개 항목으로 비교하는 예가 제시된다.

이처럼 여러 항목을 기준으로 여러 대상을 종합적으로 비교할 때는 방사형 차트가 적합하다.

| 상황                           | 적합한 그래프 |
|------------------------------|---------|
| 여러 항목으로 여러 집단 비교             | 방사형 차트  |
| 음식점별 맛, 가격, 청결도, 서비스, 분위기 비교 | 방사형 차트  |

---

### 7.2 방사형 차트 예

음식점 A, B, C를 5개 항목으로 비교하는 경우 다음과 같이 정리할 수 있다.

| 항목  | A 음식점 | B 음식점 | C 음식점 |
|-----|------:|------:|------:|
| 맛   |   4.2 |   3.8 |   4.5 |
| 가격  |   3.1 |   4.2 |   4.1 |
| 청결도 |   3.6 |   4.9 |   4.6 |
| 서비스 |   3.7 |   4.3 |   4.1 |
| 분위기 |   4.1 |   3.9 |   4.1 |

방사형 차트는 여러 평가 항목에서 어느 음식점이 상대적으로 우수한지 한눈에 비교할 수 있게 해 준다.

## 정리하기

1. 도수분포표는 겹치지 않는 몇 개의 계급에 속한 관측치 개수를 요약하여 작성한 표이다.

2. 도수분포표는 정성적 데이터뿐만 아니라 정량적 데이터에도 적용할 수 있다.

3. 정량적 데이터의 도수분포표를 작성할 때는 계급 수와 계급 폭을 세심하게 정해야 한다.

4. 히스토그램은 연속형 데이터의 특징이나 분포 모양을 살펴볼 때 사용하는 그래프이다.

5. 히스토그램은 수평축에 계급구간을 표시하고, 각 계급의 상대도수에 비례하는 넓이의 직사각형을 그린 것이다.

6. 엑셀에서 히스토그램은 [데이터 분석] 도구 또는 차트 도구를 이용하여 작성할 수 있다.

7. 산점도는 두 연속형 변수 사이의 관계를 그래프로 살펴보기 위해 사용한다.

8. 산점도는 두 변수 사이의 연관성 정도를 살펴보기 위한 분석의 첫 단계이다.

9. 엑셀에서 산점도는 [삽입] 리본 메뉴의 [차트] 그룹에서 `분산형`을 선택하여 작성한다.

10. 상관계수는 두 변수 간 선형관계의 정도를 나타내는 측도이다.

11. 상관계수는 -1과 1 사이의 값을 가진다.

12. 상관계수의 절댓값이 1에 가까울수록 두 변수는 강한 선형 연관성을 가진다.

13. 상관계수의 절댓값이 0에 가까울수록 두 변수는 약한 선형 연관성을 가진다.

14. 상관계수가 양수이면 양의 선형관계, 음수이면 음의 선형관계이다.

15. 엑셀에서 상관분석은 [데이터] 탭의 [분석] 그룹에서 [데이터 분석]을 선택한 뒤 `상관분석` 항목을 이용한다.

16. 여러 항목으로 여러 집단을 종합 비교할 때는 방사형 차트가 유용하다.

## 핵심 요약

### 1. 도수분포표

| 항목 | 핵심                         |
|----|----------------------------|
| 의미 | 계급별 관측치 개수를 요약한 표          |
| 적용 | 정성적·정량적 데이터 모두 가능          |
| 주의 | 정량적 데이터는 계급 수와 계급 폭 설정이 중요 |

---

### 2. 도수분포표 작성 공식

| 항목      | 공식                   |
|---------|----------------------|
| 계급 수    | √n ± 2~3             |
| 계급 폭    | (최대값 - 최소값) / 계급의 수  |
| 첫 하한경계값 | 최소값 - (측정의 최소단위 / 2) |
| 첫 상한경계값 | 하한경계값 + 계급의 폭        |
| 계급값     | (하한경계값 + 상한경계값) / 2  |

---

### 3. 히스토그램

| 항목    | 핵심                 |
|-------|--------------------|
| 대상    | 연속형 데이터            |
| 목적    | 분포 모양 파악           |
| 축     | 수평축에 계급구간          |
| 막대    | 도수 또는 상대도수 표현      |
| 엑셀 작성 | [데이터 분석] - [히스토그램] |

---

### 4. 산점도

| 항목    | 핵심          |
|-------|-------------|
| 대상    | 두 연속형 변수    |
| 목적    | 두 변수의 관계 확인 |
| 엑셀 차트 | 분산형         |
| 역할    | 상관분석의 첫 단계  |

---

### 5. 상관계수

| 값      | 해석         |
|--------|------------|
| r > 0  | 양의 선형관계    |
| r < 0  | 음의 선형관계    |
| r ≈ 0  | 약한 선형관계    |
| r ≈ 1  | 강한 양의 선형관계 |
| r ≈ -1 | 강한 음의 선형관계 |

## 연습문제 정리

### Q1. 다음 산점도에 대한 상관계수로 가장 적절한 것은?

선택지:

1. r = -0.87
2. r = 0.25
3. r = 0.92
4. r = 1.02

정답: 3

해설:
산점도의 점들이 오른쪽 위로 뚜렷하게 증가하는 직선 형태를 보인다. 따라서 강한 양의 상관관계를 가지며, 보기 중 가장 적절한 값은 r = 0.92이다. 상관계수는 -1과 1 사이의 값이므로 1.02는 불가능하다.

---

### Q2. 도수분포표에 대한 설명 중 옳지 않은 것은?

1. 겹치지 않는 몇 개의 범주 또는 계급에 속한 관측치의 개수를 요약하여 작성한 표이다.
2. 연속형 데이터에 대한 분석에서 도수분포표를 기초로 히스토그램을 그릴 수 있다.
3. 연속형 데이터에 대해 도수분포표를 작성할 때는 계급의 개수와 폭을 결정하는 것이 중요하다.
4. 정성적 데이터에 대한 분석에는 활용할 수 없다.

정답: 4

해설:
도수분포표는 정성적 데이터와 정량적 데이터 모두에 적용할 수 있다. 따라서 정성적 데이터에 활용할 수 없다는 설명은 옳지 않다.

---

### Q3. 두 연속형 변수 X와 Y 사이의 상관계수를 구했더니 0.9가 나왔다. 올바른 해석은?

1. X값이 커지면 Y값은 직선관계를 가지고 커진다.
2. X값이 커지면 Y값은 직선관계를 가지고 작아진다.
3. X값이 커지면 Y값은 곡선관계를 가지고 커진다.
4. X값이 커져도 Y값은 변화가 없다.

정답: 1

해설:
상관계수 0.9는 1에 가까운 양수이므로 강한 양의 선형관계를 의미한다. 따라서 X값이 커지면 Y값도 직선관계를 가지고 커지는 경향이 있다.

---

### Q4. 엑셀 함수로 이용하여 구한 시간에 흐름에 따른 전압 값을 다음과 같은 그래프로 표시하였다. 이러한 그래프를 얻기 위해 사용된 엑셀 차트 유형은?

선택지:

1. 히스토그램
2. 세로 막대형
3. 원형
4. 분산형

정답: 4

해설:
시간과 전압처럼 두 수치형 변수의 관계를 점과 선으로 표현한 그래프는 분산형 차트가 적절하다. 특히 시간 간격이나 수치 관계를 나타낼 때 분산형을 사용할 수 있다.

---

### Q5. 두 변수들 사이의 관계를 살펴보기 위해 산점도를 그리고자 한다. 선택 메뉴에서 차트 종류 입력창에서 선택할 차트 종류는?

1. 세로 막대형이나 가로 막대형
2. 영역형
3. 분산형
4. 꺾은선형

정답: 3

해설:
산점도는 엑셀 차트 종류 중 `분산형`을 선택하여 작성한다.

## 시험 직전 체크포인트

### 반드시 암기하기

- 도수분포표 = 계급별 관측치 개수 요약표
- 도수분포표는 정성적 데이터와 정량적 데이터 모두 가능
- 정량적 데이터는 계급 수와 계급 폭 결정이 중요
- 히스토그램 = 연속형 데이터의 분포 모양 확인
- 히스토그램은 계급구간 위에 직사각형을 그림
- 산점도 = 두 연속형 변수 사이의 관계를 보는 그래프
- 산점도는 상관분석의 첫 단계
- 산점도는 엑셀에서 `분산형`
- 상관계수는 -1에서 1 사이
- 상관계수의 절댓값이 1에 가까울수록 강한 선형관계
- 상관계수의 절댓값이 0에 가까울수록 약한 선형관계
- r = 0.9는 강한 양의 선형관계
- r = -0.82는 뚜렷한 음의 상관관계
- 여러 항목으로 여러 집단 비교는 방사형 차트
