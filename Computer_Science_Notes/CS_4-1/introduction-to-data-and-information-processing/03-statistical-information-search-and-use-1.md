## 1. 통계정보의 검색 및 활용

### 1.1 통계정보의 의미

통계정보는 사회, 경제, 인구, 산업, 교육 등 여러 분야의 현상을 수치로 정리한 정보이다.

통계정보는 단순한 숫자가 아니라 다음과 같은 목적으로 활용된다.

- 사회 현상 파악
- 정책 수립
- 경제 상황 분석
- 인구 변화 예측
- 지역별 특성 비교
- 합리적인 의사결정 지원

---

### 1.2 통계정보 활용의 필요성

지식정보화 사회에서는 정보를 수집하고 분석하는 능력이 중요하다.

특히 통계정보는 객관적인 수치 자료를 바탕으로 현상을 이해하고 판단하는 데 활용된다.

| 활용 목적 | 설명                           |
|-------|------------------------------|
| 현황 파악 | 현재 사회·경제·인구 상태를 수치로 확인       |
| 추세 분석 | 시간에 따른 변화 흐름을 분석             |
| 비교 분석 | 지역, 성별, 연령, 산업 등 여러 기준으로 비교  |
| 의사결정  | 정책, 경영, 연구, 학습 등에서 판단 근거로 활용 |
| 시각화   | 표, 그래프, 지도 등을 통해 쉽게 이해       |

---

## 2. 국가통계 정보

### 2.1 정부 3.0 시대

정부 3.0 시대는 정부기관 등이 보유하고 있는 공공정보를 누구나 손쉽게 활용할 수 있도록 적극적으로 개방하고 공유하는 시대를 의미한다.

주요 특징은 다음과 같다.

- 공공정보 개방
- 공공정보 공유
- 양방향 맞춤형 서비스
- 빅데이터 활용

즉, 정부기관이 가진 통계와 공공자료를 국민이 쉽게 검색하고 활용할 수 있도록 하는 것이 핵심이다.

---

### 2.2 국가통계 정보의 활용

국가통계 정보는 정부기관이나 공공기관이 작성·관리하는 통계자료이다.

대표적으로 다음과 같은 분야에서 활용된다.

| 분야 | 활용 예                  |
|----|-----------------------|
| 인구 | 장래인구추계, 고령화, 출생률, 사망률 |
| 경제 | 물가, 고용, 소비, 생산, 수출입   |
| 사회 | 교육, 복지, 보건, 문화        |
| 지역 | 지역경제, 지역인구, 지역산업      |
| 정책 | 국가정책 수립과 평가           |

## 3. 인터넷을 이용한 통계정보

### 3.1 인터넷과 통계정보

인터넷은 전 세계 수많은 컴퓨터가 연결되어 있는 네트워크이다.

인터넷은 각종 정보를 취득하는 중요한 도구이며, 지식정보화 시대에는 통계정보의 좋은 검색 도구이자 활용 도구로 이용된다.

---

### 3.2 인터넷 통계정보의 특징

| 특징      | 설명                      |
|---------|-------------------------|
| 접근성     | 누구나 인터넷을 통해 통계자료에 접근 가능 |
| 검색성     | 필요한 통계를 키워드나 분야별로 검색 가능 |
| 최신성     | 최신 통계자료를 빠르게 확인 가능      |
| 비교 가능성  | 여러 항목, 지역, 기간을 비교 가능    |
| 시각화     | 그래프, 지도 등으로 통계정보를 표현 가능 |
| 자료 다운로드 | 개인용 컴퓨터로 자료를 내려받아 활용 가능 |

---

### 3.3 GIS와 통계정보

통계정보는 지리정보시스템과 결합되어 더 유용하게 활용될 수 있다.

GIS는 Geographic Information System의 약자이다.

| 구분  | 의미                            |
|-----|-------------------------------|
| GIS | Geographic Information System |
| 한국어 | 지리정보시스템                       |
| 활용  | 통계정보를 지도, 그래프 등과 결합하여 표현      |

예를 들어 지역별 인구, 경제활동, 상권, 산업 현황 등을 지도 위에 표시하면 지역 차이를 쉽게 파악할 수 있다.

통계청 홈페이지에서는 여러 국가통계 서비스를 제공한다.

대표적인 서비스는 다음과 같다.

| 서비스           | 설명                       |

| 마이크로 데이터 서비스  | 이용자가 직접 가공할 수 있는 원시자료 제공 |

## 5. KOSIS 통계정보시스템

### 5.1 KOSIS의 의미

KOSIS는 Korea Statistical Information System의 약자이다.

KOSIS 통계정보시스템은 통계청을 비롯한 여러 통계작성기관에서 제공하는 통계정보를 제공하는 시스템이다.

---

### 5.2 KOSIS의 특징

KOSIS에는 주요 국내통계와 국제통계가 수록되어 있다.

| 구분    | 내용                              |
|-------|---------------------------------|
| 제공 기관 | 통계청 및 여러 통계작성기관                 |
| 제공 자료 | 국내통계, 국제통계                      |
| 분류 기준 | 월별, 분기별, 연도별                    |
| 분야    | 20개 분야 이상의 통계자료                 |
| 활용 방식 | 검색, 비교, 시계열 분석, 그래프 표현, 자료 다운로드 |

---

### 5.3 KOSIS 주요 기능

KOSIS를 통해 이용할 수 있는 주요 기능은 다음과 같다.

1. 통계자료의 목록, 작성기관, 수록기관, 소재정보 파악
2. 원하는 항목의 통계자료 검색
3. 여러 항목 간 통계자료 비교 검색
4. 원하는 항목의 장기 시계열 자료 검색
5. 항목의 통계자료를 크기순으로 배열
6. 통계자료를 시각적인 그래프로 표현
7. 개인용 컴퓨터로 자료 받기

---

### 5.4 KOSIS 활용 예

KOSIS에서는 다음과 같은 자료를 찾을 수 있다.

- 인구 통계
- 고용 통계
- 물가 통계
- 산업 통계
- 지역 통계
- 국제 비교 통계
- 장기 시계열 통계

시험에서는 KOSIS가 “여러 통계작성기관의 통계정보를 제공하는 통계정보시스템”이라는 점을 기억한다.

---

## 6. e-나라지표

### 6.1 e-나라지표의 의미

e-나라지표는 국정 전문 분야에 대한 각종 국정통계를 제공하는 서비스이다.

주요 제공 내용은 다음과 같다.

- 정확한 통계
- 통계의 추이
- 통계 분석
- 국정 분야별 주요 지표

---

### 6.2 e-나라지표 활용 예

e-나라지표에서는 소비자물가지수와 같은 국정 지표를 확인할 수 있다.

| 구분    | 예                |
|-------|------------------|
| 경제 지표 | 소비자물가지수, 경제성장률 등 |
| 사회 지표 | 고용, 복지, 교육 등     |
| 정책 지표 | 정부 정책과 관련된 주요 통계 |

시험에서는 “국정 전문 분야에 대한 통계와 추이, 통계분석을 제공”이라는 표현이 나오면 e-나라지표를 떠올리면 된다.

---

## 7. 마이크로 데이터 서비스

### 7.1 마이크로 데이터 서비스의 의미

마이크로 데이터 서비스는 사용자가 직접 자료를 가공하여 처리할 수 있도록 조사자료를 제공하는 서비스이다.

- 원시자료 제공
- 이용자가 직접 자료를 가공 가능
- 분석 목적에 맞게 자료 처리 가능
- 교육용 데이터 제공

---

### 7.2 마이크로 데이터의 활용

마이크로 데이터는 집계된 표 형태의 통계가 아니라, 이용자가 직접 분석할 수 있는 원자료에 가깝다.

| 구분       | 설명                       |
|----------|--------------------------|
| 일반 통계자료  | 이미 집계·정리된 통계표            |
| 마이크로 데이터 | 이용자가 직접 가공·분석할 수 있는 조사자료 |
| 활용       | 연구, 과제, 교육, 통계 분석 실습     |

시험에서는 “이용자가 직접 자료를 가공하여 이용할 수 있도록 원시자료를 제공하는 서비스”라는 표현이 나오면 마이크로 데이터 서비스를 떠올리면 된다.

---

## 8. 통계 시각화 예

통계정보는 숫자 표만으로 제공되는 것이 아니라 그래프, 지도, 상황판 등으로 시각화될 수 있다.

강의에서 제시된 통계 시각화 예는 다음과 같다.

| 예        | 설명                       |
|----------|--------------------------|
| 경제활동참가율  | 경제활동 인구의 참여 정도를 시각적으로 표현 |
| 소비자 물가지수 | 물가 변동 상황을 지표로 확인         |
| 추계인구     | 장래 인구 변화를 예측하여 표현        |
| 지역경제상황판  | 지역별 경제 상황을 종합적으로 시각화     |

## 시험 직전 체크포인트

### 반드시 암기하기

- 정부 3.0 = 공공정보 개방·공유
- 인터넷 = 통계정보 검색 및 활용 도구
- KOSIS = Korea Statistical Information System
- KOSIS = 여러 통계작성기관의 통계정보 제공
- e-나라지표 = 국정 분야 통계, 추이, 통계분석 제공
- 마이크로 데이터 서비스 = 원시자료 제공, 이용자 직접 가공
- GIS = 통계정보 + 지리정보 결합
- 통계 시각화 = 경제활동참가율, 소비자 물가지수, 추계인구, 지역경제상황판
