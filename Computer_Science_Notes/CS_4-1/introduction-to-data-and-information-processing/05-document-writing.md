## 1. 한글 2018 기초 사용법

### 1.1 한글 문서창의 구성

한글 2018 문서창은 문서를 작성하고 편집하기 위한 여러 도구로 구성된다.

| 구성 요소   | 설명                             |
|---------|--------------------------------|
| 제목 막대   | 현재 열려 있는 문서의 이름과 저장 위치 표시      |
| 메뉴 막대   | 파일, 편집, 보기, 입력, 서식 등 메뉴 제공     |
| 기본도구 막대 | 새 문서, 열기, 저장, 인쇄 등 기본 기능 제공    |
| 서식도구 막대 | 글꼴, 글자 크기, 정렬, 줄간격 등 서식 기능 제공  |
| 눈금자     | 문서 여백, 들여쓰기, 문단 폭 등을 조절하는 데 사용 |

---

### 1.2 기본도구 막대

기본도구 막대에는 자주 사용하는 명령이 아이콘 형태로 배치되어 있다.

| 번호 | 기능    |
|----|-------|
| 1  | 새 글   |
| 2  | 문서 열기 |
| 3  | 저장하기  |
| 4  | 인쇄    |
| 5  | 미리 보기 |
| 6  | 되돌리기  |
| 7  | 다시 실행 |

---

### 1.3 서식도구 모음

서식도구 모음은 문서를 작성하면서 글꼴, 글자 크기, 글자 모양, 문단 모양 또는 스타일을 지정할 수 있는 기능을 모아 놓은 것이다.

아이콘이나 드롭다운 목록 형태로 구성되어 있어 대화상자를 사용하지 않고 바로 조절할 수 있다.

| 기능    | 설명                          |
|-------|-----------------------------|
| 스타일   | 문서의 스타일 지정                  |
| 언어    | 입력 언어 설정                    |
| 글꼴    | 글자 모양 지정                    |
| 글자 크기 | 글자의 크기 지정                   |
| 진하게   | 글자를 굵게 표시                   |
| 기울임   | 글자를 기울여 표시                  |
| 밑줄    | 글자 아래 선 표시                  |
| 취소선   | 글자 가운데 선 표시                 |
| 글자색   | 글자 색 지정                     |
| 정렬    | 양쪽, 왼쪽, 가운데, 오른쪽, 배분, 나눔 정렬 |
| 줄간격   | 줄 사이의 간격 조정                 |

---

### 1.4 새로운 문서 작성

새 문서를 작성하는 기본 과정은 다음과 같다.

1. 한글 2018 실행
2. [파일] - [새 문서] 선택
3. 내용 입력
4. [파일] - [저장하기] 또는 [다른 이름으로 저장하기] 선택

#### 입력 관련 기본 키

| 키          | 기능          |
|------------|-------------|
| Space Bar  | 한 칸 띄우기     |
| Enter      | 다음 줄로 이동    |
| Delete     | 커서 뒤의 글자 삭제 |
| Back Space | 커서 앞의 글자 삭제 |

---

### 1.5 블록을 지정하여 여러 줄 지우기

여러 줄을 한꺼번에 지우려면 먼저 지우고자 하는 내용을 블록으로 지정한다.

1. 지우고자 하는 내용의 처음에 커서를 둔다.
2. 마우스 왼쪽 버튼을 누른 채 원하는 곳까지 끌어준다.
3. 선택된 범위가 검은색으로 표시된다.
4. 이 상태를 블록이라고 한다.
5. Delete 키를 눌러 한꺼번에 삭제한다.

지우기 취소는 다음 기능을 이용한다.

- [편집] - [되돌리기]
- 기본도구 모음의 되돌리기 아이콘

---

### 1.6 문자 변환

#### 한글 글자판을 영문으로 바꾸기

한글 글자판을 영문 글자판으로 변경하는 방법은 다음과 같다.

| 방법     | 설명                       |
|--------|--------------------------|
| 메뉴 이용  | [도구] - [글자판] - [글자판 바꾸기] |
| 단축키 이용 | 왼쪽 Shift + Space Bar     |
| 키보드 이용 | 한/영 키                    |

영문 입력 시 대소문자 구분은 Caps Lock 키를 이용한다.

---

### 1.7 한자 입력

한자를 입력하려면 먼저 한글로 입력한 뒤 한자로 변환한다.

| 방법     | 설명                        |
|--------|---------------------------|
| 메뉴 이용  | [입력] - [한자 입력] - [한자로 변환] |
| 단축키 이용 | F9                        |

한글 2018에서는 한 글자씩 변환과 단어 변환을 지원한다.

---

### 1.8 단축키

단축키는 메뉴 모음이나 기본도구 모음을 이용하지 않고 명령을 실행할 수 있게 하는 키 조합이다.

| 단축키      | 기능       |
|----------|----------|
| Alt + N  | 새 문서     |
| Alt + O  | 문서 열기    |
| Alt + S  | 저장 관련 기능 |
| Ctrl + X | 오려두기     |
| Ctrl + C | 복사하기     |
| Ctrl + V | 붙이기      |

예를 들어 새 문서창을 불러올 때 [파일] - [새 문서]를 이용하는 대신 Alt + N을 사용할 수 있다.

---

### 1.9 문자표 이용

특수문자는 전화기 모양, 로마자, 원 문자 등 일반적인 문자 이외의 그림이나 기호를 의미한다.

특수문자를 넣는 방법은 다음과 같다.

| 방법     | 설명           |
|--------|--------------|
| 메뉴 이용  | [입력] - [문자표] |
| 단축키 이용 | Ctrl + F10   |

예를 들어 전화기 모양 기호를 넣으려면 문자 영역에서 전각 기호를 선택하고 원하는 기호를 선택한다.

---

### 1.10 단어 검색과 변환

특정 단어가 사용된 위치를 찾거나 그 단어를 다른 단어로 바꿀 때는 찾기 또는 찾아 바꾸기 기능을 이용한다.

| 기능     | 메뉴              |
|--------|-----------------|
| 찾기     | [편집] - [찾기]     |
| 찾아 바꾸기 | [편집] - [찾아 바꾸기] |

---

### 1.11 문서 저장

문서를 저장하는 방법은 다음과 같다.

| 상황         | 방법                          |
|------------|-----------------------------|
| 현재 문서 저장   | [파일] - [저장하기]               |
| 다른 이름으로 저장 | [파일] - [다른 이름으로 저장]         |
| 다른 형식으로 저장 | [파일] - [다른 이름으로 저장]에서 형식 선택 |

---

### 1.12 한글 종료

| 상황             | 방법                           |
|----------------|------------------------------|
| 여러 문서 중 하나만 닫기 | [파일] - [문서 닫기] 또는 우측 상단 X 클릭 |
| 한글 작업 전체 종료    | [파일] - [끝]                   |

## 2. 문서의 편집 및 인쇄

### 2.1 블록 지정

문서를 복사하거나 이동하려면 먼저 블록을 지정해야 한다.

| 방법     | 설명             |
|--------|----------------|
| 마우스 끌기 | 원하는 범위를 드래그    |
| F3     | 줄 단위 블록 설정     |
| F4     | 칸 단위 블록 설정     |
| 모두 선택  | [편집] - [모두 선택] |

시험에서는 F3은 줄 단위, F4는 칸 단위 블록 설정으로 기억한다.

---

### 2.2 블록 이동과 복사

#### 블록 이동

블록으로 지정한 내용을 다른 위치로 옮기는 방법이다.

1. 이동할 범위를 블록으로 지정한다.
2. [편집] - [오려두기] 또는 Ctrl + X를 누른다.
3. 원하는 위치에 커서를 둔다.
4. [편집] - [붙이기] 또는 Ctrl + V를 누른다.

#### 블록 복사

블록으로 지정한 내용을 복사하여 다른 위치에 붙이는 방법이다.

1. 복사할 범위를 블록으로 지정한다.
2. [편집] - [복사하기] 또는 Ctrl + C를 누른다.
3. 원하는 위치에 커서를 둔다.
4. [편집] - [붙이기] 또는 Ctrl + V를 누른다.

---

### 2.3 글자 서식 변경

글자 모양을 변경하려면 원하는 범위를 블록으로 설정한 후 글자 모양 기능을 이용한다.

| 항목  | 설명                    |
|-----|-----------------------|
| 글꼴  | 신명조, 굴림체, 궁서체 등 글자 모양 |
| 장평  | 글자의 가로와 세로 비율         |
| 자간  | 글자와 글자 사이의 간격         |
| 진하게 | 글자를 굵게 표시             |
| 기울임 | 글자를 기울여 표시            |
| 밑줄  | 글자 아래 선 표시            |
| 취소선 | 글자 가운데 선 표시           |
| 글자색 | 글자의 색 지정              |

글자 모양 대화상자는 다음 방법으로 열 수 있다.

- [서식] - [글자 모양]
- 오른쪽 마우스 클릭 후 빠른 메뉴에서 [글자 모양]

---

### 2.4 장평과 자간

| 구분 | 설명                                  |
|----|-------------------------------------|
| 장평 | 글자의 가로와 세로 비율, 글자 크기에 대한 퍼센트 값으로 표시 |
| 자간 | 글자와 글자 사이의 간격, 글자 폭에 대한 비율로 표시      |

일반적인 장평은 100%로 적용된다.

---

### 2.5 문단 모양

문단 모양은 문단의 정렬, 여백, 간격, 줄 나눔 기준 등을 지정하는 기능이다.

문단 모양 대화상자는 다음과 같이 실행한다.

1. 서식을 지정할 부분을 블록으로 설정한다.
2. [서식] - [문단 모양]을 선택한다.
3. 문단 모양 대화상자에서 원하는 항목을 설정한다.

---

### 2.6 정렬 방식

정렬 방식은 문단을 구성하는 단어들을 문단의 폭에 맞게 배치하는 방법이다.

| 정렬 방식  | 설명                 |
|--------|--------------------|
| 양쪽 정렬  | 문단의 양쪽 끝을 맞춤       |
| 왼쪽 정렬  | 왼쪽 기준으로 정렬         |
| 오른쪽 정렬 | 오른쪽 기준으로 정렬        |
| 가운데 정렬 | 가운데 기준으로 정렬        |
| 배분 정렬  | 글자 사이를 벌려 문단 폭에 맞춤 |
| 나눔 정렬  | 줄 끝에 맞게 나누어 정렬     |

---

### 2.7 여백

여백은 제본 여백 이외에 문단의 폭을 결정하는 요인이다.

| 구분   | 설명                            |
|------|-------------------------------|
| 들여쓰기 | 문단의 첫 번째 줄을 오른쪽으로 밀어 쓰는 것     |
| 내어쓰기 | 첫 줄 또는 특정 부분을 왼쪽으로 당겨 빼어 쓰는 것 |

---

### 2.8 간격

간격은 줄과 줄 사이 또는 문단과 문단 사이의 거리이다.

| 구분       | 설명                   |
|----------|----------------------|
| 줄 간격     | 문단을 구성하는 줄과 줄 사이의 간격 |
| 문단 위 간격  | 문단 위쪽의 간격            |
| 문단 아래 간격 | 문단 아래쪽의 간격           |

줄 간격은 글자 크기에 따라 자동으로 변하는 퍼센트 단위로 설정할 수 있으며, 0~500% 사이에서 변경할 수 있다.

---

### 2.9 줄 나눔 기준

줄 나눔 기준은 문단 안의 한 낱말이 오른쪽 여백에 걸릴 때 어떤 기준으로 줄을 나눌지를 정하는 것이다.

| 언어 | 줄 나눔 기준        |
|----|----------------|
| 한글 | 글자 또는 어절 단위    |
| 영문 | 단어, 하이픈, 글자 단위 |

한글과 영문이 섞인 문장에서는 한글은 글자 단위, 영문은 단어 단위로 정렬하는 것이 바람직하다.

최소 공백은 낱말 사이의 빈칸 간격을 일정 범위까지 줄여 줄 끝의 낱말이 다음 줄로 넘어가지 않도록 자리를 확보해 준다.

---

### 2.10 그림 삽입

그림을 삽입하는 방법은 다음과 같다.

1. [입력] - [그림]을 선택한다.
2. 그림 넣기 대화상자에서 그림 파일을 지정한다.
3. [열기]를 클릭한다.
4. 그림을 삽입할 위치에서 원하는 크기만큼 마우스를 끌어 그림을 삽입한다.

---

### 2.11 다단 편집

다단 편집은 문서에서 세로 방향으로 공간을 나누고 각 공간마다 글을 배치하는 것이다.

설정 방법은 다음과 같다.

1. [쪽] - [단] - [다단 설정] 선택
2. 다단 대화상자에서 단 종류, 단 개수, 너비, 간격 설정

신문이나 잡지처럼 여러 단으로 나뉜 문서를 만들 때 사용한다.

---

### 2.12 쪽 나누기

쪽 나누기는 현재 쪽을 다 채우지 않고 강제로 다음 쪽으로 넘어가도록 하는 기능이다.

특히 다단 나누기를 한 경우 하나의 쪽이 채워지지 않으면 다음 쪽으로 넘어가지 않을 수 있는데, 이때 쪽 나누기를 이용한다.

설정 방법:

1. 쪽 나누기를 할 위치에 커서를 둔다.
2. [쪽] - [쪽 나누기] 선택

---

### 2.13 각주와 미주

각주와 미주는 전문적인 용어에 대한 추가 설명을 넣을 때 사용한다.

| 구분 | 위치                | 용도           |
|----|-------------------|--------------|
| 각주 | 해당 낱말이 있는 페이지 아래쪽 | 페이지 단위 설명    |
| 미주 | 문서 맨 마지막          | 문서 전체의 참고 설명 |

#### 각주 설정

1. 각주를 넣을 단어 오른쪽에 커서를 둔다.
2. [입력] - [주석] - [각주] 선택
3. 각주 설명 입력

#### 미주 설정

1. 원하는 위치에 커서를 둔다.
2. [입력] - [주석] - [미주] 선택
3. 미주 설명 입력

---

### 2.14 인쇄하기

문서를 인쇄하기 전에 편집 용지와 미리보기를 확인한다.

| 단계    | 방법             |
|-------|----------------|
| 용지 지정 | [파일] - [편집 용지] |
| 출력 확인 | [파일] - [미리보기]  |
| 인쇄    | [파일] - [인쇄]    |

미리보기는 인쇄하고자 하는 내용이 실제로 어떻게 출력될지를 보여준다.

---

## 3. 표 작성

### 3.1 표 만들기

표를 작성하는 기본 방법은 다음과 같다.

1. [입력] - [표] - [표 만들기] 선택
2. 줄 수와 칸 수 지정
3. [작성] 클릭
4. 표를 삽입할 위치에서 원하는 크기가 되도록 마우스로 조정

---

### 3.2 기존 내용을 표로 만들기

4. 표/셀 속성 대화상자에서 [글자처럼 취급] 체크

## 4. 수식 편집기 사용 방법

### 4.1 수식 편집기 실행

한글 2018에서 수식을 입력하려면 수식 편집기를 사용한다.

| 방법     | 설명              |
|--------|-----------------|
| 메뉴 이용  | [입력] - [수식]     |
| 아이콘 이용 | 기본 도구모음의 수식 아이콘 |
| 단축키 이용 | Ctrl + N, M     |

Ctrl + N, M은 Ctrl 키를 누른 채로 N과 M을 순서대로 누르는 방식이다.

---

### 4.2 수식 입력 예

예를 들어 다음 수식을 입력한다고 하자.

    x^3 - 2x^2 + 3 = 7

입력 방법:

1. 수식 편집기를 실행한다.
2. 지수 형태 아이콘을 선택한다.
3. 수식 입력창의 `{ }^{ }`에 x와 3을 순서대로 입력한다.
4. 나머지 수식을 입력한다.
5. [넣기]를 클릭한다.

---

### 4.3 지수와 제곱근 입력

| 수식 형태    | 입력 방법         |
|----------|---------------|
| 9^4      | 지수 입력 아이콘 이용  |
| 세제곱근 197 | 제곱근 입력 아이콘 이용 |

수식 편집기에서는 지수, 분수, 제곱근, 괄호, 합 기호 등을 아이콘 또는 입력 명령어로 작성할 수 있다.

---

### 4.4 수식 편집기 입력어 예

| 입력어   | 의미          |
|-------|-------------|
| chi   | 카이 기호       |
| over  | 분수 형태       |
| cases | 조건별 수식 묶음   |
| #     | 수식 안에서 줄 바꿈 |
| &     | 수식 정렬 기준    |

예를 들어 카이제곱 통계량은 수식 편집기에서 chi, over 등을 활용해 작성할 수 있다.

    chi^2 = sum { (관측도수 - 기대도수)^2 over 기대도수 }

또는 도수 기호를 사용하면 다음과 같이 표현할 수 있다.

    chi^2 = sum { (f_i - fhat_i)^2 over fhat_i }

## 정리하기

1. 한글 2018 문서창은 제목 막대, 메뉴 막대, 기본도구 막대, 서식도구 막대, 눈금자 등으로 구성된다.

2. 제목 막대에는 현재 열려 있는 문서의 이름과 저장된 위치가 나타난다.

3. 기본도구 막대에는 새 글, 문서 열기, 저장하기, 인쇄, 미리 보기, 되돌리기, 다시 실행 기능이 있다.

4. 서식도구 모음은 글꼴, 글자 크기, 글자 모양, 문단 모양, 스타일 등을 지정하는 기능을 모아 놓은 것이다.

5. 새 문서는 [파일] - [새 문서]로 만들고, 저장은 [파일] - [저장하기] 또는 [다른 이름으로 저장하기]를 이용한다.

6. 한글/영문 글자판 변경은 [도구] - [글자판] - [글자판 바꾸기], Shift + Space Bar, 한/영 키로 할 수 있다.

7. 한자 입력은 먼저 한글로 입력한 뒤 [입력] - [한자 입력] - [한자로 변환] 또는 F9 키를 이용한다.

8. 특수문자는 [입력] - [문자표] 또는 Ctrl + F10을 이용하여 넣는다.

9. 단어 검색은 [편집] - [찾기], 단어 변환은 [편집] - [찾아 바꾸기]를 이용한다.

10. 문서를 복사하거나 이동하려면 먼저 블록 설정을 해야 한다.

11. F3은 줄 단위 블록 설정, F4는 칸 단위 블록 설정에 사용한다.

12. 블록 이동은 Ctrl + X 후 Ctrl + V, 블록 복사는 Ctrl + C 후 Ctrl + V를 이용한다.

13. 글자 서식은 글꼴, 장평, 자간, 진하게, 기울임, 밑줄, 글자색 등을 설정할 수 있다.

14. 문단 모양에서는 정렬 방식, 여백, 간격, 줄 나눔 기준 등을 지정한다.

15. 다단 편집은 세로 방향으로 공간을 나누어 각 공간마다 글을 배치하는 기능이다.

16. 각주는 해당 페이지 아래쪽에, 미주는 문서 맨 마지막에 설명을 넣는 기능이다.

17. 표는 [입력] - [표] - [표 만들기]를 이용해 작성한다.

18. 표의 셀 크기는 F5로 셀을 선택한 뒤 Shift, Ctrl, Alt와 방향키 조합으로 조절할 수 있다.

19. 수식 편집기는 [입력] - [수식] 또는 Ctrl + N, M으로 실행한다.

20. 수식 편집기에서는 지수, 분수, 제곱근, 조건식 등을 입력할 수 있다.

## 연습문제 정리

### Q1. 수식편집기 창을 이용하여 다음 수식을 작성하고자 한다. 수식편집창의 입력어로 적합한 것은?

수식 예:

    chi^2 = sum { (관측도수 - 기대도수)^2 over 기대도수 }
          = sum { (f_i - fhat_i)^2 over fhat_i }

정답: 2

해설:

카이제곱 수식을 입력할 때 핵심 입력어는 다음과 같다.

| 입력어  | 의미       |
|------|----------|
| chi  | 카이 기호    |
| &    | 수식 정렬 기준 |
| over | 분수 입력    |

따라서 chi, &, over의 조합을 기억한다.

---

### Q2. 수식편집기를 이용하여 조건식 형태의 함수를 작성하고자 한다. 수식편집창의 입력어로 적합한 것은?

예:

    f(x) = {
        1, 0 ≤ x ≤ 1
        0, otherwise
    }

정답: 3

해설:

조건별 수식을 나타내기 위해서는 `cases`를 사용하고, 수식 안에서 줄을 나눌 때는 `#`을 사용한다.

| 입력어   | 의미          |
|-------|-------------|
| cases | 조건별 수식 묶음   |
| #     | 수식 안에서 줄 바꿈 |

---

### Q3. 다음은 한글 2018 사용법에 대한 설명이다. 옳은 설명으로 고른 것은?

보기:

1. 복사나 이동을 위한 블록 지정 시 F3는 칸 단위 블록 설정, F4는 줄 단위 블록 지정 키이다.
2. 신문이나 잡지 등에서 쓰는 다단 편집 기능은 제공하고 있다.
3. 그림을 삽입하여 문서를 작성할 수 있다.

정답: 3

해설:

옳은 설명은 2, 3이다.

- 1번은 틀림
- F3는 줄 단위 블록 설정
- F4는 칸 단위 블록 설정
- 한글 2018은 다단 편집 기능을 제공한다.
- 그림 삽입 기능도 제공한다.

### Q4. 다음은 한글 2018 사용법에서 한자 변환에 대한 설명이다. 옳은 설명을 모두 고른 것은?

보기:

1. 한자로 된 단어를 입력하고자 하는 경우, 먼저 한글로 해당 단어를 입력하고 [입력] - [한자로 변환]을 이용하여 변환한다.
2. F5 키를 이용하여 빈칸에 한자 입력 대화상자를 띄울 수 있다.
3. 두 글자 이상의 단어는 F9 키를 이용하여 통째로 한자로 변환시킬 수 있다.

정답: 2

해설:

옳은 설명은 1, 3이다.

- 한자 입력은 먼저 한글로 입력한 뒤 한자로 변환한다.
- 한자 변환에는 F9 키를 사용할 수 있다.
- F5는 표에서 셀 선택에 주로 사용되는 키이다.

### Q5. 다음 괄호 안에 알맞은 용어로 짝지어진 것은?

문장:

    문서를 작성하다 보면 전문적인 용어에 대해 추가적인 설명이 필요할 경우가 있다.
    이때 해당 어휘가 있는 곳의 아래쪽에 설명을 다는 것이 (ㄱ)이고,
    문서 맨 마지막에 설명을 붙이는 것이 (ㄴ)이다.

정답: 1

정답 조합:

    ㄱ: 각주
    ㄴ: 미주

해설:

각주는 해당 페이지 아래쪽에 표시되고, 미주는 문서의 맨 마지막에 표시된다.
