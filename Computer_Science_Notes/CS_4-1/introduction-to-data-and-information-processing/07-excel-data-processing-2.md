## 1. 함수 사용법

### 1.1 엑셀의 연산자

엑셀에서는 함수를 사용하여 데이터를 분석하고, 워크시트 값에 대해 여러 연산을 수행할 수 있다.

엑셀에서 사용하는 연산자는 다음과 같다.

| 연산자 종류 | 의미                      |
|--------|-------------------------|
| 산술 연산자 | 더하기, 빼기, 곱셈, 나누기, 지수 계산 |
| 비교 연산자 | 두 값의 크기나 일치 여부 비교       |
| 문자 연산자 | 두 값을 연결하여 하나의 문자값 생성    |
| 참조 연산자 | 셀 범위, 합집합, 교집합 참조       |

---

### 1.2 산술 연산자

산술 연산자는 숫자 계산에 사용된다.

| 연산자 | 의미  | 예      |
|-----|-----|--------|
| +   | 더하기 | =A1+B1 |
| -   | 빼기  | =A1-B1 |
| /   | 나누기 | =A1/B1 |
| *   | 곱셈  | =A1*B1 |
| ^   | 지수  | =A1^2  |

---

### 1.3 비교 연산자

비교 연산자는 두 값을 비교하여 참 또는 거짓을 판단할 때 사용된다.

| 연산자 | 의미     | 예       |
|-----|--------|---------|
| =   | 같다     | =A1=B1  |
| >   | 크다     | =A1>B1  |
| <   | 작다     | =A1<B1  |
| >=  | 크거나 같다 | =A1>=B1 |
| <=  | 작거나 같다 | =A1<=B1 |
| <>  | 같지 않다  | =A1<>B1 |

IF 함수에서 조건을 만들 때 비교 연산자를 자주 사용한다.

---

### 1.4 문자 연산자

문자 연산자는 `&`이다.

| 연산자 | 의미                    |
|-----|-----------------------|
| &   | 두 값을 연결하여 하나의 문자값을 산출 |

예:

    =A1&B1

A1의 값과 B1의 값을 이어 붙여 하나의 문자값으로 만든다.

---

### 1.5 참조 연산자

참조 연산자는 셀이나 범위를 지정할 때 사용한다.

| 연산자 | 의미  | 예                  |
|-----|-----|--------------------|
| :   | 범위  | =SUM(A1:A10)       |
| ,   | 합집합 | =SUM(A1:A3, B1:B5) |
| 공백  | 교집합 | =SUM(A1:A6 A5:A10) |

#### 범위 참조

    =SUM(A1:A10)

A1부터 A10까지의 값을 합산한다.

#### 합집합 참조

    =SUM(A1:A3, B1:B5)

A1부터 A3까지와 B1부터 B5까지를 함께 합산한다.

#### 교집합 참조

    =SUM(A1:A6 A5:A10)

A1:A6과 A5:A10이 겹치는 부분을 참조한다.

## 2. 함수 사용 방법

### 2.1 함수의 기능

함수는 값, 셀 참조, 다른 함수 등을 사용하여 새로운 값을 생성하는 기능이다.

엑셀 함수는 다음과 같은 작업에 활용된다.

- 합계 계산
- 평균 계산
- 조건 판단
- 분산과 표준편차 계산
- 최대값과 최소값 계산
- 데이터 분석

---

### 2.2 일반적인 함수 형식

함수식은 등호 `=`로 시작한다.

일반적인 형식은 다음과 같다.

    =함수이름(인수, 인수, ..., 인수)

예:

    =AVERAGE(A1:B3)

의미:

- `AVERAGE`는 평균을 구하는 함수이다.
- `A1:B3`은 평균을 구할 셀 범위이다.

---

### 2.3 함수 사용 예

#### 숫자를 인수로 사용하는 경우

    =SUM(2,3,4,5,5)

해석:

- 2, 3, 4, 5, 5를 모두 더한다.

#### 셀 범위를 인수로 사용하는 경우

    =SUM(A1:A5)

해석:

- A1부터 A5까지의 값을 모두 더한다.

---

## 3. 셀 참조 방법

셀 참조 방법에는 상대참조, 절대참조, 혼합참조가 있다.

---

### 3.1 상대참조

상대참조는 행 이름이나 열 이름만 사용하여 셀을 참조하는 방식이다.

| 예               | 의미            |
|-----------------|---------------|
| A1              | A열 1행         |
| D3              | D열 3행         |
| =AVERAGE(A2:G2) | A2부터 G2까지의 평균 |

상대참조는 수식을 다른 셀로 복사하면 참조 위치가 함께 변한다.

예를 들어 D2 셀에 다음 수식을 입력한다고 하자.

    =AVERAGE(B2:C2)

이 수식을 D3으로 복사하면 다음처럼 자동으로 바뀐다.

    =AVERAGE(B3:C3)

---

### 3.2 절대참조

절대참조는 참조하는 셀이나 셀 범위를 고정하기 위해 `$` 표시를 사용하는 방식이다.

| 예                   | 의미                |
|---------------------|-------------------|
| $A$1                | A열과 1행 모두 고정      |
| $D$3                | D열과 3행 모두 고정      |
| =AVERAGE($A$2:$G$2) | A2부터 G2까지의 범위를 고정 |

절대참조는 수식을 복사해도 참조 위치가 바뀌지 않는다.

---

### 3.3 혼합참조

혼합참조는 행 이름이나 열 이름 중 한쪽에만 `$` 표시를 붙여 참조하는 방식이다.

| 예                 | 의미                 |
|-------------------|--------------------|
| $A1               | A열은 고정, 행은 변함      |
| D$3               | D열은 변함, 3행은 고정     |
| =AVERAGE($A2:$G2) | 열은 고정, 행은 상대적으로 변함 |

---

### 3.4 참조 방식 비교

| 참조 방식 | 예        | 복사 시 변화          |
|-------|----------|------------------|
| 상대참조  | A1       | 행과 열이 모두 변할 수 있음 |
| 절대참조  | $A$1     | 행과 열이 모두 고정      |
| 혼합참조  | $A1, A$1 | 행 또는 열 중 하나만 고정  |

---

## 4. 함수 사용 예

### 4.1 AVERAGE 함수를 이용한 평균 계산

성적 데이터에서 각 사람의 평균점수를 구할 수 있다.

예를 들어 B열에는 통계학 점수, C열에는 수학 점수가 있다고 하자.

1. D1 셀에 `평균`이라고 입력한다.
2. D2 셀에 커서를 둔다.
3. 수식 입력줄에 다음 수식을 입력한다.

   =AVERAGE(B2:C2)

4. Enter 키를 누른다.
5. 나머지 학생은 채우기 기능을 이용한다.

---

### 4.2 채우기 기능

채우기 기능은 하나의 셀에 입력한 수식이나 값을 다른 셀로 쉽게 복사하는 기능이다.

평균 계산 예에서 D2 셀에 다음 수식을 입력한 뒤,

    =AVERAGE(B2:C2)

채우기 기능을 사용하면 D3, D4, D5 등에도 자동으로 다음과 같이 수식이 적용된다.

    =AVERAGE(B3:C3)
    =AVERAGE(B4:C4)
    =AVERAGE(B5:C5)

---

### 4.3 IF 함수 사용

IF 함수는 주어진 조건을 평가하여 조건이 참인 경우와 거짓인 경우에 서로 다른 값을 표시한다.

구문은 다음과 같다.

    =IF(주어진 조건, value if true, value if false)

| 구성 요소          | 의미                   |
|----------------|----------------------|
| 주어진 조건         | 참 또는 거짓으로 판정되는 값이나 식 |
| value if true  | 조건이 참일 때 표시할 값       |
| value if false | 조건이 거짓일 때 표시할 값      |

---

### 4.4 IF 함수 예: 합격 여부 판정

통계학 점수가 60점 이상이면 `합격`, 60점 미만이면 `불합격`으로 표시하고자 한다.

1. E1 셀에 `합격여부`라고 입력한다.
2. E2 셀에 다음 수식을 입력한다.

   =IF(B2<60,"불합격","합격")

의미:

- B2가 60보다 작으면 `불합격`
- 그렇지 않으면 `합격`

같은 의미로 다음과 같이 쓸 수도 있다.

    =IF(B2>=60,"합격","불합격")

---

## 5. 함수 마법사의 이용

### 5.1 함수 마법사의 의미

함수 마법사는 사용자가 함수를 쉽게 선택하고 인수를 입력할 수 있도록 도와주는 기능이다.

함수 이름이나 인수 형식이 익숙하지 않을 때 사용하면 편리하다.

---

### 5.2 함수 마법사 사용 절차

과목별 평균점수를 구하는 예를 기준으로 절차를 정리하면 다음과 같다.

1. 계산된 함수 값이 입력될 셀을 선택한다.
2. 함수 마법사를 클릭한다.
3. 함수 선택 대화상자에서 사용할 함수를 선택한다.
4. 함수의 인수로 사용할 셀 범위를 마우스로 드래그하여 설정한다.
5. 인수가 올바르게 지정되었는지 검토한다.
6. 확인을 선택한다.

---

## 6. 자주 사용하는 엑셀 함수

### 6.1 AVERAGE 함수

AVERAGE 함수는 인수의 산술 평균을 구한다.

형식:

    =AVERAGE(number1, number2, number3, ...)

예:

    =AVERAGE(B2:C2)

의미:

- B2부터 C2까지의 평균을 구한다.

---

### 6.2 VAR 함수

VAR 함수는 표본자료의 표본분산을 구한다.

형식:

    =VAR(number1, number2, number3, ...)

예:

    =VAR(B2:B31)

의미:

- B2부터 B31까지의 표본분산을 구한다.

---

### 6.3 STDEV 함수

STDEV 함수는 표본자료의 표준편차를 구한다.

형식:

    =STDEV(number1, number2, number3, ...)

예:

    =STDEV(B2:B31)

의미:

- B2부터 B31까지의 표준편차를 구한다.

---

### 6.4 MEDIAN 함수

MEDIAN 함수는 인수의 중앙값을 구한다.

형식:

    =MEDIAN(number1, number2, number3, ...)

예:

    =MEDIAN(B2:B31)

의미:

- B2부터 B31까지의 중앙값을 구한다.

---

### 6.5 MAX 함수

MAX 함수는 데이터의 최대값을 구한다.

형식:

    =MAX(number1, number2, number3, ...)

예:

    =MAX(B2:B31)

의미:

- B2부터 B31까지의 최대값을 구한다.

---

### 6.6 MIN 함수

MIN 함수는 데이터의 최소값을 구한다.

형식:

    =MIN(number1, number2, number3, ...)

예:

    =MIN(B2:B31)

의미:

- B2부터 B31까지의 최소값을 구한다.

---

## 7. 엑셀을 사용한 데이터 처리 예

### 7.1 기술통계량

기술통계량(descriptive statistics)은 데이터의 전반적인 분포 형태와 특성을 쉽게 파악하기 위해 데이터를 정리·요약한 수치이다.

대표적인 기술통계량은 다음과 같다.

- 평균
- 표준오차
- 중앙값
- 최빈값
- 표준편차
- 분산
- 첨도
- 왜도
- 범위
- 최소값
- 최대값
- 합
- 관측수

---

### 7.2 데이터 분석 도구

엑셀에서는 [데이터] 탭의 [분석] 그룹에 있는 [데이터 분석]을 이용하여 다양한 분석을 수행할 수 있다.

엑셀에서 제공하는 대표적인 데이터 분석 도구는 다음과 같다.

| 분석 도구   | 설명                   |
|---------|----------------------|
| 분산분석    | 집단 간 평균 차이 분석        |
| 상관분석    | 변수 간 관련성 분석          |
| 기술통계분석  | 평균, 표준편차 등 요약 통계량 계산 |
| 이동평균법   | 시계열 자료의 추세 파악        |
| 지수 평활법  | 시계열 자료의 평활화          |
| 히스토그램   | 자료의 분포 확인            |
| 난수생성    | 무작위 수 생성             |
| 순위와 백분율 | 순위와 백분율 계산           |
| 회귀분석    | 변수 간 관계 분석           |
| t-검정    | 평균 차이 검정             |
| 표본 추출   | 표본 선택                |

---

### 7.3 데이터 분석 도구가 보이지 않는 경우

[데이터] 탭의 [분석] 그룹에서 [데이터 분석]이 나타나지 않는 경우에는 분석 도구를 추가해야 한다.

절차:

1. [파일] 선택
2. [옵션] 선택
3. [추가 기능] 선택
4. [분석 도구] 선택
5. 확인 또는 실행

---

### 7.4 기술통계량 구하는 절차

엑셀에서 기술통계량을 구하는 절차는 다음과 같다.

1. [데이터] 탭 선택
2. [분석] 그룹에서 [데이터 분석] 선택
3. 분석 도구 목록에서 [기술 통계법] 선택
4. 입력범위 지정

6. 출력 범위 또는 새 워크시트 선택
7. 요약통계량 선택
8. 확인 클릭
9. 결과 확인

## 8. 실습: 성적데이터

성적데이터 실습에서는 다음 작업을 수행한다.

- 상대참조, 절대참조, 혼합참조 이용
- AVERAGE 함수를 이용한 평균 계산
- IF 함수를 이용한 합격·불합격 판정
- 데이터 분석 도구를 이용한 기술통계 분석

---

### 8.1 평균 계산 예

B2 셀과 C2 셀에 입력된 두 과목의 평균을 구하려면 다음 수식을 사용한다.

    =AVERAGE(B2:C2)

이 수식은 B2와 C2 두 셀에 입력된 값의 평균을 계산한다.

---

### 8.2 합격 여부 판정 예

B2 셀의 값이 60점 미만이면 `불합격`, 60점 이상이면 `합격`이라고 표시하려면 다음 수식을 사용한다.

    =IF(B2<60,"불합격","합격")

또는 다음과 같이 쓸 수 있다.

    =IF(B2>=60,"합격","불합격")

---

### 8.3 두 과목 점수 합계 판정 예

B2와 C2의 합이 130 이상이면 `합격`, 그렇지 않으면 `불합격`으로 표시하려면 다음 수식을 사용할 수 있다.

    =IF(SUM(B2:C2)>=130,"합격","불합격")

---

### 8.4 채우기 기능 예

D2 셀에 다음 수식을 입력했다고 하자.

    =B$2+$C$2

이 수식을 채우기 기능으로 D11 셀까지 복사하면, D3 셀의 결과는 다음과 같이 계산된다.

    B$2 + $C$2

즉, B$2와 $C$2를 참조하므로 같은 셀의 값을 사용한다.

강의 연습문제 기준 D3 셀 결과는 136이다.

## 정리하기

1. 엑셀에서는 함수를 사용하여 데이터를 분석하고, 산술 연산, 비교 연산, 문자 연산, 참조 연산 등을 수행할 수 있다.

2. 산술 연산자에는 +, -, /, *, ^가 있다.

3. 비교 연산자에는 =, >, <, >=, <=, <>가 있다.

4. 문자 연산자 `&`는 두 값을 연결하여 하나의 문자값을 만든다.

5. 참조 연산자에는 범위를 나타내는 `:`, 합집합을 나타내는 `,`, 교집합을 나타내는 공백이 있다.

6. 함수식은 등호 `=`로 시작하며, 일반적으로 `=함수이름(인수, 인수, ...)` 형식이다.

7. 상대참조는 행 이름이나 열 이름만 사용하여 셀을 참조하며, 수식을 복사하면 참조 위치가 변한다.

8. 절대참조는 `$` 기호를 사용하여 행과 열을 고정하며, 수식을 복사해도 참조 위치가 변하지 않는다.

9. 혼합참조는 행 또는 열 중 한쪽만 고정하는 방식이다.

10. AVERAGE 함수는 평균을 구한다.

11. IF 함수는 조건이 참일 때와 거짓일 때 서로 다른 값을 표시한다.

12. 함수 마법사는 함수를 쉽게 선택하고 인수를 지정할 수 있도록 도와준다.

13. VAR 함수는 표본분산, STDEV 함수는 표본표준편차, MEDIAN 함수는 중앙값, MAX 함수는 최대값, MIN 함수는 최소값을 구한다.

14. 기술통계량은 데이터의 전반적인 분포 형태와 특성을 파악하기 위해 데이터를 정리·요약한 수치이다.

15. 엑셀의 데이터 분석 도구를 이용하면 기술통계분석, 분산분석, 상관분석, 회귀분석, t-검정 등 기초적인 통계분석을 수행할 수 있다.

## 연습문제 정리

### Q1. 엑셀에서 B2 셀의 값이 60보다 작으면 “불합격”, 60 이상이면 “합격”을 D2 셀에 표시하고자 한다. 적합한 엑셀 함수식은?

1. =IF(B2<60 "합격" ELSE "불합격")
2. =IF(B2<60 "불합격" ELSE "합격")
3. =IF(B2<60, "합격", "불합격")
4. =IF(B2<60, "불합격", "합격")

정답: 4

해설:

IF 함수의 기본 구조는 다음과 같다.

    =IF(조건, 참일 때 값, 거짓일 때 값)

B2가 60보다 작으면 불합격이고, 그렇지 않으면 합격이므로 정답은 다음 수식이다.

    =IF(B2<60,"불합격","합격")

---

### Q2. B2와 C2 셀에 데이터를 입력한 후 두 값의 평균값을 구하고자 한다. 적합한 수식은?

1. AVERAGE(B2:C2)
2. =AVERAGE(B2:C2)
3. STDEV(B2:C2)
4. =STDEV(B2:C2)

정답: 2

해설:

엑셀 함수식은 반드시 등호 `=`로 시작한다. 평균을 구하는 함수는 AVERAGE이므로 정답은 다음과 같다.

    =AVERAGE(B2:C2)

---

### Q3. 데이터를 입력한 후 그림과 같이 각 케이스에 대하여 평균 점수를 구하고자 한다. A 부분에 적합한 수식은?

1. AVERAGE(B2:C2)
2. =AVERAGE(B2:C2)
3. MEDIAN(B2:C2)
4. =MEDIAN(B2:C2)

정답: 2

해설:

평균 점수를 구하는 함수는 AVERAGE이고, 함수식은 `=`로 시작해야 한다.

    =AVERAGE(B2:C2)

---

### Q4. D2 셀에 `=IF(SUM(B2:C2)>=130,"합격","불합격")`을 입력하였을 때 그 결과는?

선택지:

1. 140
2. 70
3. 합격
4. 불합격

정답: 3

해설:

SUM(B2:C2)의 결과가 130 이상이면 `합격`을 표시한다. 문제의 해당 데이터에서는 조건을 만족하므로 결과는 `합격`이다.

---

### Q5. D2 셀에 `=B$2+$C$2`를 입력한 후 드래그 기능을 이용하여 D11 셀까지 채워 넣었다. 이때 D3 셀의 결과는?

선택지:

1. 136
2. 121
3. 오류 발생
4. 알 수 없음

정답: 1

해설:

수식은 다음과 같다.

    =B$2+$C$2

- `B$2`는 2행이 고정된 혼합참조이다.
- `$C$2`는 C열과 2행이 모두 고정된 절대참조이다.

따라서 D3으로 복사해도 참조하는 값은 B2와 C2 기준으로 유지되어 결과는 136이다.
