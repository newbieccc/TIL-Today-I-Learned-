## 1. R 소개 및 다운로드

### 1.1 R의 소개

R은 통계분석과 그래프 작성을 위해 널리 사용되는 통계 소프트웨어 시스템이다.

| 구분           | 설명                                             |
|--------------|------------------------------------------------|
| 통계 소프트웨어 시스템 | 데이터를 효율적으로 처리하고 분석할 수 있으며, 다양한 통계분석과 그래프 구현 가능 |
| 대화형 언어       | 명령어를 입력하면 즉시 실행되는 interpreted language         |
| 객체지향 시스템     | 데이터, 변수, 행렬 등이 모두 객체로 다루어짐                     |

R에서는 객체를 생성할 때 `<-` 또는 `=` 연산자를 사용할 수 있다.

    x <- 4
    y = 4

---

### 1.2 R의 주요 기능

| 기능      | 설명                                |
|---------|-----------------------------------|
| 자료처리 기능 | 자료의 구성, 정렬, 결합 등을 프로그램으로 쉽게 처리    |
| 자료분석 기능 | 통계분석 함수와 분석 결과를 다양하게 제공           |
| 언어 기능   | 함수문 작성이 쉽고 C, FORTRAN 등과 인터페이스 가능 |
| 그래픽스 기능 | 대화형 그래픽스와 분석 결과의 그래프 처리 기능이 뛰어남   |

## 4. 그룹별 기술통계량 구하기

### 4.1 tapply 함수

`tapply()` 함수는 그룹별로 통계량을 계산할 때 사용한다.

기본 형식은 다음과 같다.

    tapply(분석할 변수, 그룹 변수, 적용할 함수)

예:

    tapply(ex8$salary, ex8$sex, mean)

의미:

- salary를 분석한다.
- sex 그룹별로 나눈다.
- 각 그룹의 평균을 구한다.

---

### 4.2 성별 월급여 평균

    tapply(ex8$salary, ex8$sex, mean)

실행 결과:

    1        2
    241.6667 245.0000

성별을 남, 여로 표시하려면 이름을 지정할 수 있다.

    mean_Sal_Sex = tapply(ex8$salary, ex8$sex, mean)
    rownames(mean_Sal_Sex) = c("남", "여")
    mean_Sal_Sex

실행 결과:

    남       여
    241.6667 245.0000

---

### 4.3 성별 월급여 표준편차

    tapply(ex8$salary, ex8$sex, sd)

실행 결과:

    1        2
    97.45084 114.45523

---

### 4.4 교육정도별 월급여 표준편차

    sd_sal_edu = tapply(ex8$salary, ex8$edu, sd)
    sd_sal_edu

실행 결과:

    1         2        3
    190.91883 61.84658 89.95369

교육정도 이름을 지정한다.

    rownames(sd_sal_edu) = c("중졸이하", "고졸", "대졸이상")
    sd_sal_edu

실행 결과:

    중졸이하      고졸    대졸이상
    190.91883  61.84658   89.95369

---

### 4.5 성별과 교육정도별 평균

성별과 교육정도 두 그룹을 함께 사용하려면 리스트를 만든다.

    sex_edu = list(ex8$sex, ex8$edu)

    mean_sal_sex_edu = tapply(ex8$salary, sex_edu, mean)
    mean_sal_sex_edu

실행 결과 예:

       1        2        3
    1 NA 173.3333 310
    2 235 290.0000 220

행과 열 이름을 지정할 수 있다.

    rownames(mean_sal_sex_edu) = c("M", "F")
    colnames(mean_sal_sex_edu) = c("Middle", "High", "College-")
    mean_sal_sex_edu

실행 결과:

        Middle    High College-
    M       NA 173.3333      310
    F      235 290.0000      220

## 8. 데이터프레임

### 8.1 데이터프레임 생성

데이터프레임은 서로 다른 형태의 변수를 하나의 표 형태로 묶은 자료구조이다.

    x = c("red", "green", "blue")
    y = c(1,2,3)

    dframe = data.frame(x,y,z)

실행 결과:

    x     y z
    1 red   1 4
    2 green 2 5
    3 blue  3 6

---

### 8.2 데이터프레임 원소 접근

| 인수      | 의미           |

### 10.2 모든 원소가 같은 행렬

         [,1] [,2] [,3]

    [2,]    1    1    1
    [3,]    1    1    1
    [4,]    1    1    1

### 10.3 서브 행렬 추출

행렬을 만든다.

    x = c(1:12)
    x = matrix(x, ncol=4, byrow=T)

1~3열을 선택한다.

    x[,c(1:3)]

실행 결과:

         [,1] [,2] [,3]
    [1,]    1    2    3
    [2,]    5    6    7

    x[c(1:3), -2]

### 10.4 행렬 관련 함수

## 정리하기

1. R은 통계분석과 그래프 구현에 적합한 통계 소프트웨어 시스템이다.

2. R은 명령어를 입력하면 즉시 실행되는 대화형 언어이다.

3. R은 객체지향 시스템으로, 데이터·변수·행렬 등이 모두 객체로 다루어진다.

4. R의 객체는 `<-` 또는 `=` 연산자를 이용하여 생성할 수 있다.

5. R은 자료처리, 자료분석, 언어, 그래픽스 기능을 제공한다.

6. 텍스트 자료는 `read.table()` 또는 `read.csv()`로 읽을 수 있다.

7. 엑셀 자료는 `xlsx` 또는 `readxl` 패키지를 이용하여 읽을 수 있다.

8. 패키지를 사용하려면 `library(패키지명)`으로 불러와야 한다.

9. `head()`는 데이터의 앞부분, 기본적으로 처음 6개 케이스를 보여준다.

10. `names()`는 데이터 객체의 변수 이름을 확인할 때 사용한다.

11. `mean()`은 평균, `sd()`는 표준편차, `summary()`는 요약통계량을 구할 때 사용한다.

12. `tapply()`는 그룹별 기술통계량을 구할 때 사용한다.

13. `table()`은 빈도표와 분할표를 만들 때 사용한다.

14. `barplot()`은 막대그림, `pie()`는 원그림, `hist()`는 히스토그램을 작성한다.

15. `stem()`은 줄기-잎 그림, `boxplot()`은 상자그림, `plot()`은 산점도 등 기본 그래프를 작성한다.

16. `factor()`는 범주형 변수에 수준과 이름을 지정할 때 사용할 수 있다.

17. 벡터는 `c()`로 만들 수 있으며, 조건을 이용하여 일부 원소를 추출할 수 있다.

18. 데이터프레임은 `data.frame()`으로 만들며, `$` 또는 `[행, 열]` 방식으로 변수를 선택할 수 있다.

19. `seq()`는 일정한 규칙을 가진 수열을 생성한다.

20. 행렬은 `matrix()`로 만들며, 행렬곱은 `%*%`, 전치행렬은 `t()`를 이용한다.

21. `apply(x, 1, mean)`은 행 평균, `apply(x, 2, mean)`은 열 평균을 구한다.

### Q1. 데이터 객체의 처음 6개의 케이스를 출력하고자 한다. R 명령은?

### Q2. 데이터 객체 `ex.data`의 변수들을 직접 사용하고자 한다. 유용한 명령은?

선택지:

1. load(ex.data)
2. detach(ex.data)
3. unload(ex.data)
4. attach(ex.data)

정답: 4

해설:
`attach()`는 데이터프레임의 변수명을 직접 사용할 수 있도록 해 준다. 예를 들어 `ex.data$salary` 대신 `salary`처럼 사용할 수 있다.

---

### Q3. 그룹변수인 성별(sex)에 따라 변수 salary의 평균(mean)을 구하는 명령은?

4. tapply(mean, sex, salary)

그룹별 통계량은 `tapply(분석변수, 그룹변수, 함수)` 형식으로 구한다.

    tapply(salary, sex, mean)
