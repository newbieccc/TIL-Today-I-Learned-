## 1. 함수의 기능과 형식

### 1.1 엑셀의 채우기 기능

엑셀의 채우기 기능은 연속된 숫자나 일정한 규칙으로 변화하는 숫자를 쉽게 입력할 수 있도록 하는 기능이다.

예를 들어 1, 2를 입력한 뒤 채우기 핸들을 이용하면 3, 4, 5, ...처럼 연속된 숫자를 자동으로 입력할 수 있다.

---

### 1.2 엑셀 함수의 기능

엑셀 함수는 값, 셀 참조, 다른 함수 등을 사용하여 새로운 값을 생성한다.

| 구분   | 설명                         |
|------|----------------------------|
| 값    | 숫자, 문자, 날짜 등 직접 입력한 값      |
| 셀 참조 | A1, B2, C3처럼 셀 주소를 이용      |
| 함수   | SUM, AVERAGE, DATE, LEFT 등 |
| 결과   | 계산이나 처리 결과로 새로운 값 생성       |

---

### 1.3 엑셀 함수의 형식

엑셀 함수의 기본 형식은 다음과 같다.

    =함수이름(인수, 인수, ..., 인수)

예:

    =SUM(A1:A3)

의미:

- `=`: 수식의 시작
- `SUM`: 함수 이름
- `A1:A3`: 함수에 전달되는 인수

---

### 1.4 함수 사용 규칙

함수를 사용할 때는 다음 규칙을 따른다.

| 규칙           | 설명                              |
|--------------|---------------------------------|
| 등호로 시작       | 모든 수식은 `=`로 시작                  |
| 괄호 필요        | 함수 이름 다음에는 인수를 묶는 괄호가 필요        |
| 다양한 인수 사용 가능 | 숫자, 셀 범위, 논리값, 문자값, 다른 함수 사용 가능 |
| 인수 구분        | 인수가 여러 개이면 콤마 `,`로 구분           |

---

### 1.5 함수 사용 예

#### 셀 범위를 인수로 사용하는 경우

    =SUM(A1:A3)

A1부터 A3까지의 값을 더한다.

#### 숫자를 직접 인수로 사용하는 경우

    =SUM(2,3,4,5,5)

2, 3, 4, 5, 5를 모두 더한다.

#### 다른 함수를 인수로 사용하는 경우

    =SUM(5+2, AVERAGE(5,7), 10)

5+2의 결과, AVERAGE(5,7)의 결과, 10을 모두 더한다.

## 2. 엑셀 함수 활용 사례

## 2.1 채우기 기능과 함수 사용 예제

### 2.1.1 문제 상황

오늘 1원을 저금하고, 내일은 그 2배, 그 다음 날은 전날의 2배를 저금한다고 하자.

이렇게 저금하면 며칠째 되는 날에 누적액이 1억 원을 돌파하는지 계산한다.

---

### 2.1.2 입력 구조

| 셀  | 입력 내용  |
|----|--------|
| A1 | 날짜     |
| B1 | 저금액    |
| C1 | 누적 저금액 |

---

### 2.1.3 날짜 입력

1. A2 셀에 1 입력
2. A3 셀에 2 입력
3. 자동 채우기 기능을 이용해 30까지 입력

---

### 2.1.4 저금액과 누적 저금액 계산

첫째 날:

    B2 = 1
    C2 = B2

둘째 날:

    B3 = 2*B2
    C3 = C2+B3

이후 B3:C3 범위를 선택한 뒤 채우기 핸들을 드래그하여 30일째 되는 날까지 채운다.

---

### 2.1.5 결과

누적액이 처음으로 1억 원을 넘는 날짜는 27일째이다.

즉, 1원에서 시작해서 매일 2배씩 저금하면 27일째에 누적 저금액이 1억 원을 초과한다.

---

## 2.2 SUM 함수를 이용한 평점 평균 구하기

### 2.2.1 교과목 성적 등급과 평점

| 등급 |  A+ |  A0 |  A- |  B+ |  B0 |  B- |  C+ |  C0 |  C- |  D+ |  D0 |  D- | F |
|----|----:|----:|----:|----:|----:|----:|----:|----:|----:|----:|----:|----:|--:|
| 평점 | 4.3 | 4.0 | 3.7 | 3.3 | 3.0 | 2.7 | 2.3 | 2.0 | 1.7 | 1.3 | 1.0 | 0.7 | 0 |

---

### 2.2.2 성적표 예

| 과목명        | 신청 학점 | 성적 |
|------------|------:|----|
| 경영학개론      |     3 | B0 |
| 세계의 역사     |     3 | A- |
| 컴퓨터의 이해    |     3 | A0 |
| 데이터정보처리 입문 |     3 | A+ |
| 인터넷서비스     |     3 | B+ |
| 계          |    15 |    |

---

### 2.2.3 평점 평균 계산 방식

평점 평균은 각 과목의 학점과 평점을 곱한 뒤, 전체 학점으로 나누어 계산한다.

    평점 평균 = Σ(학점 × 평점) / Σ학점

---

### 2.2.4 SUM 함수를 이용한 계산 절차

1. 각 과목의 성적에 대응하는 평점을 입력한다.
2. E1 셀에 `학점*평점`을 입력한다.
3. E2 셀에 다음 수식을 입력한다.

   =B2*D2

4. E2 셀을 선택하고 E6 셀까지 채우기 기능을 사용한다.
5. E7 셀에 다음 수식을 입력한다.

   =SUM(E2:E6)/B7

이 수식은 학점과 평점의 곱을 모두 더한 뒤 전체 학점으로 나누어 평점 평균을 구한다.

---

## 2.3 SUMPRODUCT 함수

### 2.3.1 SUMPRODUCT 함수의 의미

SUMPRODUCT 함수는 배열의 대응되는 값끼리 곱한 뒤 그 합을 구하는 함수이다.

즉, 다음과 같은 계산을 한 번에 수행한다.

    첫 번째 배열 × 두 번째 배열 → 각각 곱함 → 곱한 값들을 합산

---

### 2.3.2 SUMPRODUCT를 이용한 평점 평균 계산

학점 범위가 B2:B6이고, 평점 범위가 D2:D6일 때 평점 평균은 다음과 같이 구할 수 있다.

    =SUMPRODUCT(B2:B6,D2:D6)/B7

의미:

- B2:B6의 학점과 D2:D6의 평점을 각각 곱한다.
- 곱한 값을 모두 더한다.
- 전체 학점 B7로 나눈다.

---

### 2.3.3 SUMPRODUCT 활용 예

A1:A3과 B1:B3에 다음 값이 있다고 하자.

| 셀  | 값 |
|----|--:|
| A1 | 1 |
| A2 | 2 |
| A3 | 3 |
| B1 | 3 |
| B2 | 2 |
| B3 | 1 |

수식:

    =SUMPRODUCT(A1:A3,B1:B3)

계산:

    1×3 + 2×2 + 3×1
    = 3 + 4 + 3
    = 10

## 3. 날짜/시간 및 문자함수 이용

## 3.1 날짜/시간 함수

### 3.1.1 DATE 함수

DATE 함수는 특정 날짜의 일련 번호를 구해 주는 함수이다.

형식:

    =DATE(year, month, day)

예:

    =DATE(2026,5,18)

의미:

- 2026년 5월 18일에 해당하는 날짜값을 생성한다.

---

### 3.1.2 YEAR 함수

YEAR 함수는 날짜에 해당하는 연도를 표시한다.

형식:

    =YEAR(날짜 또는 숫자)

예:

    =YEAR(A1)

A1 셀에 날짜가 들어 있다면 해당 날짜의 연도를 추출한다.

---

### 3.1.3 NOW 함수

NOW 함수는 현재 시점을 나타내는 날짜/시간 함수이다.

형식:

    =NOW()

특징:

- 인수가 필요 없다.
- 현재 날짜와 시간을 반환한다.
- 입력하거나 다시 계산하는 시점마다 값이 변한다.

---

## 3.2 텍스트 함수

### 3.2.1 LEFT 함수

LEFT 함수는 문자열의 첫 문자부터 지정된 개수만큼 문자를 표시한다.

형식:

    =LEFT(text, num_chars)

예:

    =LEFT(A1,2)

A1 셀에 `대한민국`이 입력되어 있다면 결과는 다음과 같다.

    대한

---

### 3.2.2 RIGHT 함수

RIGHT 함수는 문자열의 마지막 문자부터 지정된 개수만큼 문자를 표시한다.

형식:

    =RIGHT(text, num_chars)

예:

    =RIGHT("Sale Price",5)

결과:

    Price

---

### 3.2.3 MID 함수

MID 함수는 문자열의 지정한 위치부터 지정한 개수만큼 문자를 표시한다.

형식:

    =MID(text, start_num, num_chars)

예:

    =MID("대한민국",3,2)

결과:

    민국

또 다른 예:

A2 셀에 `대한민국`이 입력되어 있을 때,

    =MID(A2,2,2)

결과:

    한민

---

## 4. 날짜/시간 및 문자함수 이용 예제

### 4.1 예제 목표

각 사람의 주민등록번호와 입사일로부터 다음 값을 계산한다.

- 생년월일
- 나이
- 근속년수

또한 월급은 통화표시 `₩`를 넣고 3자리마다 단위를 표시한다.

---

### 4.2 생년월일 계산

주민등록번호에서 앞 6자리를 이용하여 생년월일을 계산한다.

G1 셀에 `생년월일`을 입력하고, G2 셀에 다음 수식을 입력한다.

    =DATE(LEFT(C2,2), MID(C2,3,2), MID(C2,5,2))

의미:

| 함수          | 역할                      |
|-------------|-------------------------|
| LEFT(C2,2)  | 주민등록번호 앞 2자리에서 연도 추출    |
| MID(C2,3,2) | 3번째 자리부터 2자리에서 월 추출     |
| MID(C2,5,2) | 5번째 자리부터 2자리에서 일 추출     |
| DATE        | 추출한 연, 월, 일을 날짜 형식으로 변환 |

---

### 4.3 나이 계산

H1 셀에 `나이`를 입력하고, H2 셀에 다음 수식을 입력한다.

    =YEAR(NOW())-YEAR(G2)

의미:

- 현재 연도에서 생년월일의 연도를 뺀다.
- 현재 시점의 나이를 대략적으로 구한다.

---

### 4.4 근속년수 계산

I2 셀에 다음 수식을 입력한다.

    =YEAR(NOW())-YEAR(D2)

의미:

- 현재 연도에서 입사일의 연도를 뺀다.
- 대략적인 근속년수를 계산한다.

---

### 4.5 통화 서식 지정

월급을 통화표시 `₩`와 3자리 단위 구분으로 표시하려면 다음 절차를 따른다.

1. F2:F6 셀 범위를 선택한다.
2. [홈] 탭의 [표시 형식] 그룹을 클릭한다.
3. 셀 서식 대화상자에서 `통화` 서식을 선택한다.
4. 기호를 `₩`로 선택한다.
5. 확인을 누른다.

---

### 4.6 표 테두리 지정

표를 보기 좋게 완성하려면 서식 도구모음에서 테두리를 선택하여 적용한다.

- 표 범위 선택
- 테두리 메뉴 선택
- 원하는 테두리 형태 적용

### Q1. A2 셀에 `대한민국`이라고 입력되어 있다. `=MID(A2,2,2)`의 결과로 알맞은 것은?

선택지:

1. 대
2. 대한
3. 한민
4. 민국

정답: 3

해설:

MID 함수는 지정한 위치부터 지정한 개수만큼 문자를 추출한다.

    =MID(A2,2,2)

A2의 문자열이 `대한민국`이면 두 번째 글자인 `한`부터 2글자를 추출한다.

결과:

    한민

---

### Q2. G2 셀에 어떤 사람의 생년월일이 `1980-12-5`로 입력되어 있다. 엑셀 함수식 `=YEAR(NOW())-YEAR(G2)`의 결과를 바르게 설명한 것은?

선택지:

1. 현재 시점의 연도가 표시된다.
2. 현재 시점의 나이를 구한다.
3. 위의 함수식에 NOW는 인수가 있기 때문에 잘못된 함수식이다.
4. 이 사람의 생년월일 중에서 연도에 해당하는 1980이 표시된다.

정답: 2

해설:

`YEAR(NOW())`는 현재 연도를 구한다.  
`YEAR(G2)`는 G2 셀의 생년월일에서 연도 1980을 추출한다.

따라서 다음 수식은 현재 연도에서 출생 연도를 빼므로 현재 시점의 나이를 구하는 식이다.

    =YEAR(NOW())-YEAR(G2)

### Q3. 학과코드의 첫 번째 자리 알파벳이 A이면 통계데이터과학과, B이면 컴퓨터과학과 학생이다. 학과 셀에 학과를 표기하고자 한다. C2 셀에 알맞은 함수식은?

정답:

    =IF(LEFT(A2,1)="A","통계데이터과학과","컴퓨터과학과")

해설:

- `LEFT(A2,1)`은 A2 셀의 첫 번째 문자를 추출한다.
- 첫 글자가 A이면 `통계데이터과학과`를 표시한다.
- 그렇지 않으면 `컴퓨터과학과`를 표시한다.

### Q4. 최종 점수는 시험점수와 과제물, 태도점수의 평균에서 결석일수에 5를 곱한 수를 뺀 것으로 계산한다. H2 셀에 알맞은 함수식은?

정답:

    =SUM(D2:F2)/3-5*G2

해설:

- D2:F2는 시험점수, 과제물, 태도점수이다.
- 세 점수의 평균은 `SUM(D2:F2)/3`이다.
- 결석일수 G2에 5를 곱한 값을 감점한다.

따라서 최종 점수는 다음과 같다.

    세 점수 평균 - 결석일수 × 5

### Q5. 다음과 같이 워크시트에 값이 입력되어 있을 때 `=SUMPRODUCT(A1:A3,B1:B3)`의 계산값은?

자료:

| 셀  | 값 |
|----|--:|
| A1 | 1 |
| A2 | 2 |
| A3 | 3 |
| B1 | 3 |
| B2 | 2 |
| B3 | 1 |

선택지:

1. 10
2. 12
3. 14
4. 16

정답: 1

해설:

SUMPRODUCT 함수는 대응되는 값끼리 곱한 뒤 합을 구한다.

    =SUMPRODUCT(A1:A3,B1:B3)

계산:

    1×3 + 2×2 + 3×1
    = 3 + 4 + 3
    = 10

## 정리하기

1. 엑셀의 채우기 기능은 연속된 숫자나 일정한 규칙으로 변화하는 숫자들을 쉽게 입력할 수 있도록 한다.

2. 엑셀 함수는 값, 셀 참조, 함수 등을 사용하여 새로운 값을 생성한다.

3. 엑셀 함수식은 등호 `=`로 시작하고, 함수 이름 뒤에는 인수를 묶는 괄호가 필요하다.

4. 함수의 인수는 숫자, 셀 범위, 논리값, 문자값, 다른 함수가 될 수 있다.

5. 인수가 여러 개 사용되는 함수는 콤마 `,`로 구분한다.

6. SUM 함수는 지정한 값이나 셀 범위의 합계를 구한다.

7. 채우기 기능을 이용하면 일정한 규칙의 값을 반복적으로 입력하거나 수식을 복사할 수 있다.

8. 평점 평균은 학점과 평점의 곱을 모두 더한 뒤 전체 학점으로 나누어 계산한다.

9. SUMPRODUCT 함수는 배열의 대응되는 값끼리 곱해서 합을 구한다.

10. DATE 함수는 연, 월, 일을 이용하여 날짜값을 만든다.

11. YEAR 함수는 날짜에서 연도를 추출한다.

12. NOW 함수는 현재 날짜와 시간을 표시하며, 인수가 필요 없다.

13. LEFT 함수는 문자열의 왼쪽부터 지정된 개수의 문자를 추출한다.

14. RIGHT 함수는 문자열의 오른쪽부터 지정된 개수의 문자를 추출한다.

15. MID 함수는 문자열의 지정한 위치부터 지정된 개수의 문자를 추출한다.

16. DATE, LEFT, MID 함수를 함께 사용하면 주민등록번호에서 생년월일을 만들 수 있다.

17. YEAR, NOW 함수를 사용하면 현재 연도를 기준으로 나이나 근속년수를 계산할 수 있다.

18. 통화 서식을 이용하면 월급을 `₩` 기호와 3자리 단위 구분 형식으로 표시할 수 있다.

## 핵심 요약

### 1. 함수 기본 형식

    =함수이름(인수, 인수, ..., 인수)

| 규칙 | 핵심                        |
|----|---------------------------|
| 시작 | 등호 `=`로 시작                |
| 괄호 | 함수 이름 뒤 괄호 필요             |
| 인수 | 숫자, 셀 범위, 논리값, 문자값, 함수 가능 |
| 구분 | 인수가 여러 개이면 콤마로 구분         |

---

### 2. 주요 함수 정리

| 함수         | 형식                               | 기능            |
|------------|----------------------------------|---------------|
| SUM        | =SUM(범위)                         | 합계 계산         |
| SUMPRODUCT | =SUMPRODUCT(배열1, 배열2)            | 대응값 곱의 합      |
| DATE       | =DATE(year, month, day)          | 날짜값 생성        |
| YEAR       | =YEAR(날짜)                        | 연도 추출         |
| NOW        | =NOW()                           | 현재 날짜와 시간     |
| LEFT       | =LEFT(text, num_chars)           | 왼쪽부터 문자 추출    |
| RIGHT      | =RIGHT(text, num_chars)          | 오른쪽부터 문자 추출   |
| MID        | =MID(text, start_num, num_chars) | 지정 위치부터 문자 추출 |
| IF         | =IF(조건, 참, 거짓)                   | 조건에 따른 값 표시   |

---

### 3. 날짜/시간 함수

| 함수   | 기억할 점            |
|------|------------------|
| DATE | 연, 월, 일로 날짜 생성   |
| YEAR | 날짜에서 연도만 추출      |
| NOW  | 현재 날짜와 시간, 인수 없음 |

---

### 4. 텍스트 함수

| 함수    | 예                      | 결과    |
|-------|------------------------|-------|
| LEFT  | =LEFT("대한민국",2)        | 대한    |
| RIGHT | =RIGHT("Sale Price",5) | Price |
| MID   | =MID("대한민국",3,2)       | 민국    |
| MID   | =MID("대한민국",2,2)       | 한민    |

### 5. 자주 나오는 수식

| 목적               | 수식                                        |
|------------------|-------------------------------------------|
| 누적 저금액           | =C2+B3                                    |
| 전날의 2배 저금        | =2*B2                                     |
| 학점×평점            | =B2*D2                                    |
| 평점 평균            | =SUM(E2:E6)/B7                            |
| SUMPRODUCT 평점 평균 | =SUMPRODUCT(B2:B6,D2:D6)/B7               |
| 생년월일 추출          | =DATE(LEFT(C2,2),MID(C2,3,2),MID(C2,5,2)) |
| 나이 계산            | =YEAR(NOW())-YEAR(G2)                     |
| 근속년수 계산          | =YEAR(NOW())-YEAR(D2)                     |
| 최종점수 계산          | =SUM(D2:F2)/3-5*G2                        |
| 학과명 표시           | =IF(LEFT(A2,1)="A","통계데이터과학과","컴퓨터과학과")   |

## 시험 직전 체크포인트

### 반드시 암기하기

- 함수식은 `=`로 시작한다.
- 함수 이름 뒤의 괄호는 반드시 필요하다.
- 인수가 여러 개이면 콤마로 구분한다.
- SUM은 합계를 구한다.
- SUMPRODUCT는 대응되는 값끼리 곱한 뒤 합한다.
- DATE는 날짜값을 만든다.
- YEAR는 날짜에서 연도를 추출한다.
- NOW는 현재 날짜와 시간을 반환하며 인수가 없다.
- LEFT는 왼쪽에서 문자 추출.
- RIGHT는 오른쪽에서 문자 추출.
- MID는 지정 위치에서 문자 추출.
- 통화 서식은 `₩` 표시와 3자리 단위 구분에 사용한다.

| 문제 표현                   | 정답 후보             |

| 특정 날짜의 일련 번호            | DATE              |
| 날짜에서 연도 추출              | YEAR              |
| 현재 날짜와 시간               | NOW               |
| 인수가 필요 없는 날짜/시간 함수      | NOW               |
| 왼쪽부터 문자 추출              | LEFT              |
| 오른쪽부터 문자 추출             | RIGHT             |
| 지정 위치부터 문자 추출           | MID               |
