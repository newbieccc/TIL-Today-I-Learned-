## 1. 엑셀이란 무엇인가?

엑셀은 행과 열로 구성된 데이터를 입력, 계산, 분석, 정리, 시각화할 수 있는 스프레드시트 프로그램이다.

기본적으로 대차대조표, 성적표 작성, 고객 관리, 재고 관리 등과 같이 행과 열로 구성된 데이터 분석에 유용하다.

---

### 1.1 스프레드시트 프로그램의 기본 기능

| 기능     | 설명                           |
|--------|------------------------------|
| 데이터 입력 | 행과 열로 구성된 표 형태로 데이터 입력       |
| 수식 계산  | 사용자가 수식을 정의하여 계산 가능          |
| 함수 활용  | 엑셀 함수를 활용하여 데이터 분석 가능        |
| 보고서 작성 | 다양한 서식을 적용하여 보기 좋은 보고서 작성 가능 |

---

### 1.2 데이터의 통계분석 기능

엑셀은 데이터 분석을 위한 다양한 통계분석 도구를 제공한다.

- 다양한 통계분석 도구를 데이터 분석에 쉽게 활용 가능
- 연립방정식 풀이 가능
- 최적화 문제 해결 기능 활용 가능
- 실제 업무상의 여러 문제를 효과적으로 해결 가능

---

### 1.3 그래프 표현 기능

엑셀에서는 다양한 그래프를 손쉽게 작성할 수 있다.

| 그래프 종류  | 활용           |
|---------|--------------|
| 원 그래프   | 구성비 표현       |
| 막대 그래프  | 항목 간 크기 비교   |
| 꺾은선 그래프 | 시간에 따른 변화 표현 |

데이터 분석 결과와 함께 알맞은 그래프를 제시하면 결과를 더 효과적으로 전달할 수 있다.

---

### 1.4 데이터베이스 관리 도구 기능

엑셀은 간단한 데이터베이스 관리 도구로도 활용할 수 있다.

| 기능     | 설명                   |
|--------|----------------------|
| 정렬     | 데이터를 특정 기준에 따라 정리    |
| 자동필터   | 조건에 맞는 데이터만 표시       |
| 보고서 작성 | 데이터를 정리하여 보고서 형태로 표현 |
| 피벗 테이블 | 데이터를 손쉽게 정리하고 요약     |

## 2. 엑셀 기초 활용

### 2.1 엑셀의 시작과 종료

#### 엑셀 시작

    작업표시줄 [시작] 단추 → [Excel]

#### 엑셀 종료

    [파일] 메뉴 → [닫기]

또는 제목 표시줄의 오른쪽에 있는 닫기 단추를 클릭한다.

---

### 2.2 엑셀 화면 구성

엑셀 화면은 다음과 같은 요소로 구성된다.

| 구성 요소   | 설명                                          |
|---------|---------------------------------------------|
| 파일 메뉴   | 새 통합문서, 열기, 저장, 인쇄, 닫기, 종료 등 제공             |
| 제목 표시줄  | 현재 사용 중인 통합문서의 이름 표시                        |

| 워크시트    | 데이터를 입력하고 계산 결과가 표시되는 작업공간                  |
| 워크시트 탭  | 시트 이름 표시 및 시트 이동                            |
| 상태 표시줄  | 엑셀의 현재 상태 표시                                |

## 3. 엑셀의 기본 용어

### 3.1 파일 메뉴

파일 메뉴는 통합문서와 관련된 기본 작업 명령을 제공한다.

- 새 통합문서
- 통합문서 열기
- 저장
- 인쇄
- 닫기
- 프로그램 종료

---

### 3.2 제목 표시줄

제목 표시줄에는 현재 사용 중인 통합문서의 이름이 나타난다.

---

### 3.3 리본 메뉴

리본 메뉴는 기존 메뉴와 도구 모음의 역할을 대신한다.

| 구성         | 설명                                   |
|------------|--------------------------------------|
| 리본 메뉴 탭    | 파일, 홈, 삽입, 페이지 레이아웃, 수식, 데이터, 검토, 보기 |
| 그룹         | 각 리본 메뉴 탭의 명령을 기능별로 묶은 것             |
| 대화상자 확장 단추 | 더 자세한 설정 창을 여는 단추                    |

---

### 3.4 삽입 탭

[삽입] 탭에는 워크시트에 삽입할 수 있는 명령이 있다.

| 삽입 대상  | 설명               |
|--------|------------------|
| 피벗 테이블 | 데이터를 요약·분석       |
| 표      | 데이터 범위를 표 형태로 구성 |
| 그림     | 이미지 삽입           |
| 차트     | 데이터를 그래프로 표현     |

---

### 3.5 워크시트

워크시트(worksheet)는 데이터를 입력하고 계산 결과가 표시되는 작업공간이다.

- 열은 A, B, C, ...로 표시
- 행은 1, 2, 3, ...으로 표시
- 셀은 열과 행이 만나는 칸
- 활성 셀은 현재 선택된 셀
- 셀 참조는 열과 행의 조합으로 표현

예:

    C열 5행의 셀 주소 = C5

---

### 3.6 수식 입력 줄

수식 입력 줄은 활성 셀의 정보를 보여준다.

- 데이터를 입력할 수 있음
- 데이터를 수정할 수 있음
- 함수식을 입력할 수 있음
- 선택된 셀의 내용을 확인할 수 있음

---

### 3.7 워크시트 탭

워크시트 탭은 워크시트 아래쪽에 위치한다.

- 각 워크시트의 이름 지정 가능
- 원하는 워크시트 이름을 클릭하여 이동 가능

---

### 3.8 상태 표시줄

상태 표시줄은 엑셀 화면의 가장 아래쪽에 있으며 엑셀의 현재 상태를 표시한다.

---

## 4. 통합문서와 워크시트

### 4.1 통합문서

통합문서(workbook)는 데이터 입력이나 분석 작업을 하고 그 결과를 저장하는 하나의 파일이다.

- 하나의 엑셀 파일
- 여러 개의 워크시트로 구성 가능
- Sheet1, Sheet2, Sheet3처럼 표시되거나 사용자가 이름을 지정할 수 있음

---

### 4.2 워크시트

워크시트(worksheet)는 행과 열의 구조를 갖는 많은 셀로 구성된 작업공간이다.

통합문서와 워크시트의 관계는 다음과 같다.

| 구분   | 비유         |
|------|------------|
| 통합문서 | 한 권의 책     |
| 워크시트 | 책 안의 한 페이지 |

워크시트의 크기는 다음과 같다.

| 구분 | 범위                   |
|----|----------------------|
| 열  | A부터 XFD까지, 총 16,384개 |
| 행  | 1부터 1,048,576까지      |

---

## 5. 데이터 입력

### 5.1 데이터 관련 기본 용어

성적 데이터처럼 번호, 통계학 점수, 수학 점수 등이 있는 자료를 생각할 수 있다.

| 용어  | 의미               |
|-----|------------------|
| 변수  | 조사단위로부터 측정되는 속성  |
| 케이스 | 조사단위에 대한 정보의 집합체 |
| 변수명 | 변수의 이름           |

예를 들어 성적 데이터에서 `통계학`, `수학`은 변수명이고, 각 학생 한 명의 정보는 하나의 케이스가 된다.

---

### 5.2 워크시트에서 직접 데이터 입력

워크시트에서 직접 데이터를 입력하는 방법은 다음과 같다.

1. 데이터를 입력할 셀을 선택한다.
2. 데이터를 입력한다.
3. Enter 키를 누른다.

---

### 5.3 텍스트 파일 불러오기

이미 만들어진 텍스트 파일을 엑셀로 불러올 수 있다.

예를 들어 한글 2018을 이용해 성적 데이터를 작성하고 `성적데이터.txt`라는 텍스트 파일로 저장한 후, 엑셀에서 불러올 수 있다.

---

### 5.4 엑셀에서 텍스트 파일 불러오기 절차

#### 1단계: 파일 열기

    [파일] 메뉴 → [열기]

- 불러올 텍스트 파일이 있는 위치를 찾는다.
- 파일 이름을 지정한다.
- 파일 형식을 텍스트 파일로 지정하면 텍스트 파일을 찾기 쉽다.

#### 2단계: 텍스트 마법사 절차 따르기

텍스트 파일을 선택하면 텍스트 마법사 대화상자가 나타난다.

- 전체 3단계 절차를 따라 진행
- 구분 기호나 데이터 형식 등을 지정
- 작성된 텍스트 파일을 워크시트로 불러올 수 있음

#### 3단계: 결과 확인

텍스트 파일의 내용이 엑셀 워크시트에 표 형태로 표시된다.

---

## 6. 데이터의 이동과 복사

### 6.1 원하는 영역 선택 방법

데이터를 이동하거나 복사하려면 먼저 원하는 셀 또는 범위를 선택해야 한다.

| 선택 방법           | 설명                         |
|-----------------|----------------------------|
| 단일 셀 선택         | 원하는 셀로 이동하여 마우스 클릭         |
| 이웃한 여러 셀 선택     | 시작 셀에서 마지막 셀까지 드래그         |
| 이웃하지 않은 여러 셀 선택 | Ctrl 키를 누른 상태에서 원하는 셀들을 클릭 |
| 전체 행 선택         | 행 머리글 클릭                   |
| 전체 열 선택         | 열 머리글 클릭                   |

---

### 6.2 이웃한 셀 선택

이웃한 여러 개의 셀을 선택하려면 원하는 범위의 시작 셀에서 드래그한다.

- 시작 셀에서 마지막 셀까지 선택
- 선택 범위는 직사각형으로 표시됨

---

### 6.3 이웃하지 않은 셀 선택

이웃하지 않은 여러 셀을 선택하려면 Ctrl 키를 사용한다.

    Ctrl 키를 누른 상태에서 원하는 셀들을 클릭

시험에서는 “이웃하지 않은 셀들을 선택”이라고 나오면 Ctrl 키를 떠올린다.

---

### 6.4 행과 열의 삽입 및 삭제

행이나 열을 삽입·삭제하는 방법은 다음과 같다.

1. 행 머리글 또는 열 머리글을 클릭하여 선택한다.
2. 마우스 오른쪽 버튼을 누른다.
3. [삽입] 또는 [삭제] 명령을 선택한다.

---

### 6.5 통합문서 저장과 출력

#### 저장 방법

데이터를 입력하거나 수정하는 과정에서는 틈틈이 저장하는 것이 안전하다.

- 빠른 실행 도구 모음의 저장 단추 클릭
- 또는 [파일] - [저장] 이용

#### 출력 방법

- [파일] 단추 클릭
- [미리보기] 또는 [인쇄] 선택

---

## 7. 함수 사용법

### 7.1 엑셀의 연산자

엑셀에서는 함수를 사용하여 워크시트 데이터를 분석하고, 워크시트 값에 대해 여러 연산을 수행할 수 있다.

연산자의 종류는 다음과 같다.

| 연산자 종류 | 설명                   |
|--------|----------------------|
| 산술 연산자 | 더하기, 빼기, 곱셈, 나누기, 지수 |
| 비교 연산자 | 두 값의 크기나 일치 여부 비교    |
| 문자 연산자 | 두 값을 연결하여 하나의 문자값 생성 |
| 참조 연산자 | 셀 범위, 합집합, 교집합 참조    |

---

### 7.2 산술 연산자

| 연산자 | 의미  |
|-----|-----|
| +   | 더하기 |
| -   | 빼기  |
| /   | 나누기 |
| *   | 곱셈  |
| ^   | 지수  |

예:

    =A1+B1
    =A1-B1
    =A1/B1
    =A1*B1
    =A1^2

---

### 7.3 비교 연산자

| 연산자 | 의미     |
|-----|--------|
| =   | 같다     |
| >   | 크다     |
| <   | 작다     |
| >=  | 크거나 같다 |
| <=  | 작거나 같다 |
| <>  | 같지 않다  |

예:

    =A1=B1
    =A1>B1
    =A1<>B1

---

### 7.4 문자 연산자

문자 연산자는 `&`이다.

| 연산자 | 의미                   |
|-----|----------------------|
| &   | 두 값을 연결하여 하나의 문자값 산출 |

예:

    =A1&B1

---

### 7.5 참조 연산자

참조 연산자는 셀이나 범위를 지정할 때 사용한다.

| 연산자 | 의미  | 예                  |
|-----|-----|--------------------|
| :   | 범위  | =SUM(A1:A10)       |
| ,   | 합집합 | =SUM(A1:A3, B1:B5) |
| 공백  | 교집합 | =SUM(A1:A6 A5:A10) |

#### 범위 참조

    =SUM(A1:A10)

A1부터 A10까지의 범위를 합산한다.

#### 합집합 참조

    =SUM(A1:A3, B1:B5)

A1부터 A3까지와 B1부터 B5까지를 함께 합산한다.

#### 교집합 참조

    =SUM(A1:A6 A5:A10)

두 범위가 겹치는 부분을 참조한다.

---

## 8. 함수 사용 방법

### 8.1 함수의 기능

함수는 값, 셀 참조, 함수 등을 사용하여 새로운 값을 생성한다.

- 산술 계산
- 통계 계산
- 조건 판단
- 문자열 처리
- 데이터 분석

---

### 8.2 일반적인 함수 형식

함수식은 등호 `=`로 시작한다.

일반적인 형식은 다음과 같다.

    =함수이름(인수, 인수, ..., 인수)

예:

    =AVERAGE(A1:B3)

---

### 8.3 함수 사용 예

#### 숫자를 인수로 사용하는 경우

    =SUM(2,3,4,5,5)

#### 셀 범위를 인수로 사용하는 경우

    =SUM(A1:A5)

#### 평균 계산

    =AVERAGE(A1:B3)

## 9. 셀 참조 방법

셀 참조 방법에는 상대참조, 절대참조, 혼합참조가 있다.

---

### 9.1 상대참조

상대참조는 행 이름이나 열 이름만 사용하여 셀을 참조하는 방법이다.

| 예               | 설명           |
|-----------------|--------------|
| A1              | A열 1행        |
| D3              | D열 3행        |
| =AVERAGE(A2:G2) | A2부터 G2까지 평균 |

상대참조는 수식을 복사하면 참조 위치가 상대적으로 변경된다.

---

### 9.2 절대참조

절대참조는 참조하는 셀이나 셀 범위를 표시할 때 `$` 기호를 사용하는 방법이다.

| 예                   | 설명              |
|---------------------|-----------------|
| $A$1                | A열 1행을 고정       |
| $D$3                | D열 3행을 고정       |
| =AVERAGE($A$2:$G$2) | A2부터 G2까지 범위 고정 |

절대참조는 수식을 복사해도 참조 위치가 바뀌지 않는다.

---

### 9.3 혼합참조

혼합참조는 행 이름이나 열 이름의 한쪽에만 `$` 표시를 붙여 참조하는 방법이다.

| 예                 | 설명                 |
|-------------------|--------------------|
| $A1               | A열은 고정, 행은 변함      |
| D$3               | D열은 변함, 3행은 고정     |
| =AVERAGE($A2:$G2) | 열은 고정, 행은 상대적으로 변함 |

## 10. 실습: 성적데이터

성적데이터 실습에서는 다음 내용을 활용한다.

- 함수 사용 방법
- 상대참조
- 절대참조
- 혼합참조

예시 데이터는 번호, 통계학, 수학 점수로 구성된다.

| 변수명 | 의미     |
|-----|--------|
| 번호  | 학생 번호  |
| 통계학 | 통계학 점수 |
| 수학  | 수학 점수  |

성적 데이터에서 평균, 합계, 조건 비교 등을 수행할 때 셀 참조와 함수를 활용할 수 있다.

## 정리하기

1. 엑셀은 행과 열로 구성된 데이터를 입력, 분석, 계산, 관리, 시각화할 수 있는 스프레드시트 프로그램이다.

2. 엑셀은 대차대조표, 성적표, 고객 관리, 재고 관리 등과 같은 행과 열 구조의 데이터 분석에 유용하다.

3. 엑셀은 사용자가 수식을 정의할 수 있고, 엑셀 함수를 활용하여 분석할 수 있다.

4. 엑셀은 데이터의 통계분석, 그래프 표현, 데이터베이스 관리 도구 기능을 제공한다.

5. 데이터베이스 관리 기능에는 정렬, 자동필터, 보고서 작성, 피벗 테이블 기능 등이 있다.

6. 통합문서는 데이터 입력이나 분석 작업 결과를 저장하는 하나의 파일이다.

7. 워크시트는 행과 열로 구성된 작업공간이며 통합문서의 일부이다.

8. 통합문서는 한 권의 책에 해당하고, 워크시트는 한 페이지에 해당한다.

9. 워크시트의 열은 A부터 XFD까지 총 16,384개이고, 행은 1부터 1,048,576까지 가능하다.

10. 셀은 열과 행이 만나는 칸이며, 셀 참조는 열과 행의 조합으로 나타낸다.

11. 수식 입력 줄은 활성 셀의 정보를 보여주고, 데이터를 입력하거나 수정할 수 있으며, 함수식을 입력할 수 있다.

12. 변수는 조사단위로부터 측정되는 속성이고, 케이스는 조사단위에 대한 정보의 집합체이며, 변수명은 변수의 이름이다.

13. 텍스트 파일은 [파일] - [열기]를 통해 불러오고, 텍스트 마법사의 3단계 절차를 따라 워크시트에 입력할 수 있다.

14. 이웃하지 않은 복수의 셀을 선택할 때는 Ctrl 키를 누른 상태에서 원하는 셀을 선택한다.

15. 엑셀의 연산자에는 산술 연산자, 비교 연산자, 문자 연산자, 참조 연산자가 있다.

16. 함수식은 등호로 시작하며, 일반적으로 `=함수이름(인수, 인수, ...)` 형식이다.

17. 상대참조는 셀 주소가 수식 복사에 따라 변하고, 절대참조는 `$`를 사용하여 행과 열을 고정한다.

18. 혼합참조는 행 또는 열 중 하나만 고정하는 참조 방식이다.

### Q1. 다음은 엑셀의 통합문서와 워크시트에 대한 내용이다. 잘못된 것은?

워크시트는 고유한 이름을 가질 수 있으며, 사용자가 이름을 변경할 수 있다. 따라서 “워크시트는 항상 고정된 이름을 가져서 변경할 수 없다”는 설명은 잘못된 설명이다.

### Q2. 다음은 엑셀에서 함수식 사용방법에 대한 설명이다. 옳은 설명끼리 짝지어진 것은?

보기:

가. 값, 셀 참조, 함수 등을 사용하여 새로운 값을 생성한다.  
나. 함수식은 등호(=)로 시작한다.  
다. 엑셀에서 함수 이름은 대문자로만 입력해야 한다.

    가, 나
