# 스프링 입문 스프링부트

## 스프링 DB 접근 기술

### # H2 데이터베이스 설치

- [H2 데이터베이스 다운로드 링크](https://www.h2database.com/html/download-archive.html)
- 1.4.200 버전 다운받기 (Platform-Independent Zip으로)
- 압축해제 후 `./h2/bin`으로 들어간 후 콘솔 창에서 `./h2.bat` 실행하기
- 브라우저가 열리고 연결 누르기

### # 테이블 생성하기

- 연결 들어간 후 아래 명령문 실행

```SQL
create table member
(
 id bigint generated by default as identity,
 name varchar(255),
 primary key (id)
);
```
